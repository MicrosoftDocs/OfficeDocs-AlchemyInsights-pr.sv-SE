# <a name="migrating-to-sharepoint-online-via-migration-manager"></a>Migrera till SharePoint Online via Migreringshanteraren

Migreringshanteraren finns i det moderna SharePoint Admin Center och vägleder dig genom konfigurationen av dina klienter och skapandet av dina uppgifter. Du kan ange inställningar för global eller aktivitetsnivå, Visa alla aktivitetsförlopp och hämta aggregerade sammanfattnings-och aktivitetsnivå rapporter.

* [Kom igång med Migreringshanteraren](https://docs.microsoft.com/sharepointmigration/mm-get-started)
* [Konfigurera Migreringshanterarens klienter](https://docs.microsoft.com/sharepointmigration/mm-setup-clients)
* [Inställningar för migreringshanteraren](https://docs.microsoft.com/sharepointmigration/mm-settings)
