# <a name="removing-the-last-on-premises-exchange-server"></a>Ta bort sista lokal Exchange server

Innan du tar bort den sista Exchange-servern från organisationen på lokaler, du bör du förstå innebörden av detta för att hjälpa dig om du behöver den sista servern för andra ändamål. För mer information, Läs [hur och när du ska inaktivera lokal Exchange-servrar i en distribution av hybrid](https://technet.microsoft.com/en-us/library/dn931280%28v=exchg.150%29.aspx).
