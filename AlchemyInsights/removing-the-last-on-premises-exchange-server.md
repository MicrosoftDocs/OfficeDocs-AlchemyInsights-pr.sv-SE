# <a name="removing-the-last-on-premises-exchange-server"></a>Ta bort den sista lokala Exchange-servern

Innan du tar bort den senaste Exchange-servern från din lokala organisation bör du se till att du förstår konsekvenserna av att göra detta för att hjälpa dig att avgöra om du behöver den sista servern för andra ändamål. Mer information finns i [Så här inaktiverar du lokala Exchange-servrar i en hybriddistribution](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
