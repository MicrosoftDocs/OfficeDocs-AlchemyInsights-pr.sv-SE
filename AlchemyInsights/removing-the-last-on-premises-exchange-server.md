# <a name="removing-the-last-on-premises-exchange-server"></a>Ta bort den senaste lokala Exchange-servern

Innan du tar bort den senaste Exchange-servern från din lokala organisation bör du se till att du förstår följderna av detta för att hjälpa dig att avgöra om du behöver den senaste servern för något annat ändamål. Mer information finns [i så här tar du reda på hur och när du ska inaktivera lokala Exchange-servrar i en hybrid distribution](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
