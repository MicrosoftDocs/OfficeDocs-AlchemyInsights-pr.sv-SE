# <a name="how-to-select-all-in-outlookcom"></a>"Markera alla" i Outlook.com

Du kan bara markera alla om alla meddelanden är synliga. Detta förhindrar oavsiktliga åtgärder på meddelanden som du inte kan se.

**CTRL + klicka** väljer flera enskilda meddelanden.

**Skift + klicka** markerar alla meddelanden mellan det första och sista meddelandet du väljer.
