# <a name="how-to-select-all-in-outlookcom"></a>Så här "väljer du alla" i Outlook.com

Du kan bara markera alla om alla meddelanden är synliga. Detta för att förhindra oavsiktliga åtgärder på meddelanden som du inte kan se.

**CTRL + klicka på** markerar flera enskilda meddelanden.

**SKIFT + klicka** markerar alla meddelanden mellan det första och det sista meddelandet du väljer.
