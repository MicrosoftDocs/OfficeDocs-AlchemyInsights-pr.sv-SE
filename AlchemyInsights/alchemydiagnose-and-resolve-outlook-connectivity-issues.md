# <a name="diagnose-and-resolve-outlook-connectivity-issues"></a>Diagnosticera och lösa problem med Outlook-anslutningen

För att kunna diagnostisera och åtgärda problem med Outlook-anslutningen kan du köra [verktyget Outlook-anslutning](https://aka.ms/SaRA-OutlookDisconnect-Alchemy) på den påverkade datorn.
  
Det här verktyget utför diagnostiska kontroller och föreslår eventuella lösningar som du kan använda för att åtgärda problem med Outlook-anslutningen.
