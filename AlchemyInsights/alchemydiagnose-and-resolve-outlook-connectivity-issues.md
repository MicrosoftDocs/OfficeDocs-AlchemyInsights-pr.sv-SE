# <a name="diagnose-and-resolve-outlook-connectivity-issues"></a>Diagnostisera och lösa anslutningsproblem i Outlook

Du kan köra [Outlook anslutningsverktyg](https://aka.ms/SaRA-OutlookDisconnect-Alchemy) på datorn för att diagnostisera och åtgärda anslutningsproblem i Outlook. 
  
Det här verktyget kommer att utföra diagnostiska kontroller och föreslå möjliga lösningar för att adressera anslutningsproblem i Outlook.
