# <a name="empty-the-deleted-items-folder"></a>Tömma mappen Borttaget

1. Öppna mappen Borttaget och välj **Töm mappen**.
2. Markera **återställnings bara objekt** högst upp på sidan och välj **Töm mappen** igen.

**Det går inte att tömma mappen Borttaget**

- Om du har tagit bort många e-postmeddelanden kan det ta lite tid innan objekten flyttas eller tas bort från mappen Borttaget. Lämna webbläsarfönstret öppet medan borttagningen sker.
- Om e-postmeddelandet ändå inte tas bort korrekt väljer du ett mindre kommando som ska tas bort.
- Om borttagen e-post kommer tillbaka kan det bero på att du har tagit bort många e-postmeddelanden. Försök igen senare.
