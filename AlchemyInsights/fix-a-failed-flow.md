# <a name="fix-a-flow-that-failed"></a>Åtgärda ett flöde som misslyckades

1. Gå till [Microsoft Flow](https://flow.microsoft.com/)och klicka på ikonen Meddelanden längst upp till höger. (Eller i mobilappen väljer du fliken Aktivitet.) Välj det flöde som inte kunde titta på detaljerna.
    
2. Hitta steget med den röda utropstecknet för att se felmeddelandet för flödet.
    
3. Beroende på felmeddelandet bör du kunna redigera flödet och åtgärda det.
