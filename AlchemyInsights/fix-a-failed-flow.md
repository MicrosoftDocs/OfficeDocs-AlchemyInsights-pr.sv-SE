# <a name="fix-a-flow-that-failed"></a>Åtgärda ett flöde som misslyckades

1. Gå till [Microsoft-flödet](https://flow.microsoft.com/)och klicka på ikonen meddelanden i det övre högra hörnet. (Eller välj fliken aktivitet i mobile app.) Välj dataflödet som inte gick att visa informationen.
    
2. Hitta steg med rött utropstecken ikonen så visas felmeddelandet för flödet.
    
3. Beroende på felmeddelandet, bör du kunna redigera flödet och åtgärda. 
    
[Vanliga fel i flödet](https://go.microsoft.com/fwlink/?linkid=872110)
