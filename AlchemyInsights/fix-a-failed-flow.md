# <a name="fix-a-flow-that-failed"></a>Åtgärda ett flöde som misslyckades

1. Gå till [Microsoft Flow](https://flow.microsoft.com/)och klicka på ikonen meddelanden uppe till höger. (Eller Välj fliken aktivitet i mobilappen.) Välj det flöde som inte kunde titta på detaljerna.
    
2. Leta reda på steget med det röda utropstecknet för att se felmeddelandet för flödet.
    
3. Beroende på felmeddelandet, bör du kunna redigera flödet och åtgärda det.
