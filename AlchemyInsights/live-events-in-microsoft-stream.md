# <a name="live-events-in-microsoft-stream"></a>Live-evenemang i Microsoft Stream

Du kan skapa livehändelser med Microsoft Stream i hela organisationen. Du kan schemalägga, producera och leverera livehändelser för en mängd olika scenarier, till exempel företagsomfattande händelser, ledarskapsuppdateringar med mera.

- [Skapa en livehändelse i Microsoft Stream](https://docs.microsoft.com/stream/live-create-event)
- [Komma igång med Live-evenemang i Microsoft Teams](https://support.office.com/article/get-started-with-microsoft-teams-live-events-d077fec2-a058-483e-9ab5-1494afda578a)
- [Konfigurera kodare för livestreaming i Microsoft Stream](https://docs.microsoft.com/stream/live-encoder-setup)
