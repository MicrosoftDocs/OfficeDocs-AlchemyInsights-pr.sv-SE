# <a name="full-name-field-not-supported-with-unified-inteface-apps"></a>Fullständigt namn fält stöds inte med enhetligt gränssnitt apps

Åtgärder eller villkor som använder ett fält för **Fullständigt namn** (fullname) stöds inte i apps baserat på ett enhetligt gränssnitt. Alternativt kan du använda åtgärder eller villkor med **Förnamn** (firstname) och **Efternamn** (lastname) fält.
