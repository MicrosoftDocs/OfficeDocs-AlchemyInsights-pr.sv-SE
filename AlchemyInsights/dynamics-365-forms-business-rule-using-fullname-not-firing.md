# <a name="full-name-field-not-supported-with-unified-inteface-apps"></a>Fullständigt namn fält stöds inte med Unified inteface-appar

Åtgärder eller villkor som använder ett **fullständigt namn** (fullname)-fält stöds inte i appar som baseras på det enhetliga gränssnittet. Du kan också använda åtgärder eller villkor med **förnamn** (förnamn) och efternamn ( **efter** namn) fält.
