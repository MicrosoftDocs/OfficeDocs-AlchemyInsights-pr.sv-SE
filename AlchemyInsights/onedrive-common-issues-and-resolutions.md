# <a name="onedrive-common-issues-and-resolutions"></a>Vanliga problem och lösningar för OneDrive

Här är några vanliga avsnitt om OneDrive-problem:

- **Konfigurera OneDrive-synkronisering**: [Synkronisera dina OneDrive för företag-filer](https://go.microsoft.com/fwlink/?linkid=533375) eller [Synkronisera SharePoint-filer](https://go.microsoft.com/fwlink/?linkid=871666) med OneDrive-synkroniseringsklienten för Windows.  Använder du Mac? Du kan också [Synkronisera filer med OneDrive](https://support.office.com/article/Sync-files-with-the-OneDrive-sync-client-on-Mac-OS-X-d11b9f29-00bb-4172-be39-997da46f913f) .

- **Felsöka OneDrive-synkronisering**: om OneDrive inte synkroniserar filer på det sätt som du förväntar dig kan du [Felsöka OneDrive-synkroniseringsproblem](https://go.microsoft.com/fwlink/?linkid=866431) för Windows. Använder du Mac? Du kan [Felsöka synkroniseringsproblem](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756) också.
- **Begränsningar i OneDrive**: Lär dig mer om [begränsningar och begränsningar](https://support.office.com/article/Invalid-file-names-and-file-types-in-OneDrive-OneDrive-for-Business-and-SharePoint-64883a5d-228e-48f5-b3d2-eb39e07630fa) för fil namn och filtyper när du synkroniserar med OneDrive.
- **OneDrive-lagring**: Lär dig hur du [ändrar en viss användares OneDrive-lagringsutrymme](https://docs.microsoft.com/onedrive/change-user-storage) eller [hanterar lagrings utrymmet för OneDrive för företag](https://support.office.com/article/Manage-your-OneDrive-for-Business-storage-31519161-059C-4764-B6F8-F5CD29F7FE68).
- **Gransknings loggar**: se hur du kan [använda gransknings loggs ökning](https://docs.microsoft.com/microsoft-365/compliance/search-the-audit-log-in-security-and-compliance#search-the-audit-log) för att kontrol lera aktivitet i filer, mappar eller användare.
