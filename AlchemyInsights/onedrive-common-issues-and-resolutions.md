# <a name="onedrive-common-issues-and-resolutions"></a>OneDrive vanliga problem och lösningar

Det verkar som om du nyfiken på OneDrive. Vi kan hjälpa till att ge dig ett bättre svar om du förtydliga din fråga med en eller två meningar som beskriver ditt problem - t ex fel och funktioner som du använder för en kort stund...  

Under tiden är här några vanliga begärda avsnitt:

- **Konfigurera OneDrive-synkronisering**: [synkronisering din OneDrive för Business-filer](https://go.microsoft.com/fwlink/?linkid=533375) eller [synkronisera SharePoint-filer](https://go.microsoft.com/fwlink/?linkid=871666) med OneDrive sync-klient för Windows.  Med hjälp av en Mac? Du kan [Synkronisera filer med OneDrive](https://support.office.com/article/Sync-files-with-the-OneDrive-sync-client-on-Mac-OS-X-d11b9f29-00bb-4172-be39-997da46f913f) samt med den.

- **Felsöka OneDrive-synkronisering**: om OneDrive inte synkronisera filer som förväntat, kan du [Felsöka synkroniseringsproblem för OneDrive](https://go.microsoft.com/fwlink/?linkid=866431) för Windows. Med hjälp av en Mac? Du kan [Felsöka synkroniseringsproblem](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756?ui=en-US&rs=en-US&ad=US) för det också.
- **Gränser för OneDrive**: Lär dig mer om [restriktioner och begränsningar](https://support.office.com/article/Invalid-file-names-and-file-types-in-OneDrive-OneDrive-for-Business-and-SharePoint-64883a5d-228e-48f5-b3d2-eb39e07630fa) som gäller för filnamn och filtyper när synkronisering med OneDrive.
- **OneDrive-lagring**: se hur du [ändrar en användare OneDrive-lagringsutrymme](https://docs.microsoft.com/onedrive/change-user-storage) eller [Hantera dina OneDrive för Business-lagring](https://support.office.com/article/Manage-your-OneDrive-for-Business-storage-31519161-059C-4764-B6F8-F5CD29F7FE68).
- **Granskningsloggar**: se hur du kan [använda Granska loggen Sök](https://docs.microsoft.com/office365/securitycompliance/search-the-audit-log-in-security-and-compliance#search-the-audit-log) om du vill kontrollera aktiviteten för filer, mappar, användare och behörigheter.
