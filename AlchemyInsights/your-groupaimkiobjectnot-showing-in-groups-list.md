# <a name="your-group-aimkiobject-not-showing-in-groups-list"></a>Gruppen "{AimKIObject}" visas inte i listan grupper

Distributions gruppen "{AimKIObject}" skapades {AimKITimeElapsed} minuter sedan. Det kan ta upp till 60 minuter innan den nya gruppen visas i administrations centret.
  
Om du vill lägga till medlemmar eller göra andra ändringar kan du använda funktionen grupper i [administrations centret för Exchange](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1&amp;mkt=en-US.aspx).
