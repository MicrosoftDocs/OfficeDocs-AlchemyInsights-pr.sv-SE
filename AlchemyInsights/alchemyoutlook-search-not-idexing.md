# <a name="outlook-search-not-indexing"></a>Outlook-sökningen indexerar inte

Vi är medvetna om sökproblem i Outlook som rör de senaste säkerhetsuppdateringarna för Windows.
  
När du söker i Outlook kan du få något av dessa fel: **Något gick fel och sökningen kunde inte slutföras**eller **sökresultaten kan vara ofullständiga eftersom objekten fortfarande indexeras**. Korrigeringen utfärdades som en valfri Windows-uppdatering. Installera den som motsvarar den version av Windows som du ser det här problemet på: 
  
> [Windows 7 - 27 juni 2017 uppdatering](https://support.microsoft.com/kb/4022168.aspx)
    
> [Windows 8.1 - 27 juni 2017 uppdatering](https://support.microsoft.com/kb/4022720.aspx)
    
> [Uppdatering av Windows 10 Version 1703 – 27 juni 2017](https://support.microsoft.com/kb/4022716.aspx)
    
Den senaste informationen finns i artikeln: [Kända problem i säkerhetsuppdateringarna i juni 2017](https://support.office.com/article/Outlook-known-issues-in-the-June-2017-security-updates-3F6DBFFD-8505-492D-B19F-B3B89369ED9B.aspx) Problem #5.
