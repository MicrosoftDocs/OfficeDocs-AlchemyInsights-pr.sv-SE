# <a name="seeing-a-false-alert"></a>Visa en falsk avisering

- Skicka en fil till Microsoft för analys. Mer information finns i [Vad du ska göra med falska positiva identifieringar/negativ i Microsoft Defender Antivirus](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#submit-a-file-to-microsoft-for-analysis).
- Skapa en indikator "Tillåt" för att förhindra en falsk positiv varning. Mer information finns i [Skapa indikatorer för filer](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/indicator-file).  
- Konfigurera undantag på enheten för att förhindra att ett objekt genomsöks. Mer information finns i [Definiera ett undantag på en enskild Windows-enhet för att förhindra att ett objekt genomsöks](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#define-an-exclusion-on-an-individual-windows-device-to-prevent-an-item-from-being-scanned).
