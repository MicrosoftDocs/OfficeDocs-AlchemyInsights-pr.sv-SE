# <a name="remove-a-hub-site"></a>Ta bort en navplats

Avregistrera en navplats i nya SharePoint administratörscenter
  
1. Klicka på **aktiva platser**under **platser**i nya SharePoint administratörscenter.

2. Markera platsen och klicka på **navplatsen**Klicka **Unregister som navplats**.

3. Klicka på **OK**.
