# <a name="delete-a-contact"></a>Ta bort en kontakt

1. Överst på sidan Välj app launcher ![The App Launcher-knappen. ](media/9634bec0-78d1-4282-8aea-7c5e81f162d2.png) och välj **personer**. 
    
2. Välj den kontakt som du vill ta bort och välj **Ta bort**![bort](media/deb47846-8483-4f9d-813a-fc8fe288b583.png).
    
    Om du inte kan välja **bort**, kanske kontakten från Skype eller ett konto med anslutna sociala nätverk. Om du vill ta bort kontakten går du till Skype eller kontot där kontakten finns.
    
3. Välj **Ta bort** för att bekräfta.
