# <a name="global-reader"></a>Global läsare

Användare i den här rollen kan läsa inställningar och administrativ information för Microsoft 365-tjänster men kan inte vidta hanteringsåtgärder. Global läsare är den skrivskyddade motsvarigheten till global administratör.
Administrationscentret för OneDrive stöder inte rollen Global reader.

Mer information om global läsare och dess aktuella begränsningar finns i [Global Reader](https://docs.microsoft.com/azure/active-directory/users-groups-roles/directory-assign-admin-roles#global-reader).
