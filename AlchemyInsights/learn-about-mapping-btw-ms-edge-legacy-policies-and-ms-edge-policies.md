# <a name="learn-about-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>Lär dig mer om mappning mellan befintliga och Microsoft Edge-principer

Grup principerna för Microsoft Edge Legacy (version 45 och tidigare) och inställningarna för hantering av mobila enheter (MDM) kopplas till relevanta principer för Microsoft Edge version 80. Mer information finns i [Microsoft Edge Legacy to the policy mappning för Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2141665).
