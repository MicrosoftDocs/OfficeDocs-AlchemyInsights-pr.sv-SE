# <a name="site-is-not-available"></a>Webbplatsen är inte tillgänglig

Om webbplatsen SharePoint eller OneDrive inte är tillgängliga för flera användare, kan det finnas ett problem med tjänsten tillfälligt. Kontrollera att [tjänsten hälsa instrumentpanelen](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) för att se om din hyresgäst påverkas.
