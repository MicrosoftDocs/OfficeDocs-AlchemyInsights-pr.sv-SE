# <a name="site-is-not-available"></a>Webbplats finns inte

Om din SharePoint-eller OneDrive-webbplats inte är tillgänglig för flera användare kan det finnas ett tillfälligt tjänstproblem. Kontrollera den [service Health instrumentpanelen](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) för att se om din klient påverkas.
