# <a name="site-is-not-available"></a>Webbplatsen är inte tillgänglig

Om din SharePoint- eller OneDrive-webbplats inte är tillgänglig för flera användare kan det bero på ett tillfälligt serviceproblem. Kontrollera [instrumentpanelen för tjänsthälsa](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) för att se om din klient påverkas.
