# <a name="site-is-not-available"></a>Webbplatsen är inte tillgänglig

Om din SharePoint-eller OneDrive-webbplats inte är tillgänglig för flera användare kan det finnas ett tillfälligt tjänst problem. Kontrol lera [instrument panelen för tjänstens hälsa](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) för att se om din klient organisation påverkas.
