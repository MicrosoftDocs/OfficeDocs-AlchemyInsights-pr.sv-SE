# <a name="migrate-public-folder-data-to-microsoft-365"></a>Migrera data från en offentlig mapp till Microsoft 365

Om du inte har många gemensamma mappar att ta med till Microsoft 365 är det enklast att använda dem för att kopiera data till. PST-filer och importera dem sedan till Microsoft 365. För små mängder data är det enkelt att dra och släppa en offentlig mapp i en. PST-fil kan vara tillräcklig. Om du har mer data än så (upp till 30 GB) har vi dokumenterat en [process](https://technet.microsoft.com/library/dn874017%28v=exchg.150%29.aspx) som du kan använda för att exportera mappar till PST-filer.
  
För större organisationer som flyttar gemensamma mappar till Microsoft 365 finns det guider:
  
- [Migrera äldre gemensamma mappar](https://docs.microsoft.com/exchange/collaboration-exo/public-folders/batch-migration-of-legacy-public-folders) (Exchange 2010)

- [Migrera gemensamma mappar i Exchange 2013, Exchange 2016 eller Exchange 2019](https://docs.microsoft.com/Exchange/collaboration/public-folders/migrate-to-exchange-online)

Du kan också migrera gemensamma [mappar till Microsoft 365-grupper.](https://docs.microsoft.com/exchange/collaboration-exo/public-folders/migrate-your-public-folders-to-microsoft-365-groups)
