# <a name="resolving-issues-sharing-onenote-notebooks"></a>Lösa problem med att dela OneNote-anteckningsböcker

Knappen Dela finns längst upp till höger i OneNote.

- Skriv in en e-postadress för att dela anteckningsboken.
- Klicka på **Dela**.

Stäng och öppna anteckningsboken i OneNote igen för att börja dela igen.
