# <a name="change-your-profile-name-in-outlookcom"></a>Ändra ditt profil namn i Outlook.com

1. Gå till [ditt Microsoft-konto](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. Välj **Redigera namn**.
3. Ange förnamn och efter namn och välj **Spara**.

Låt upp till 24 timmar behöva ändra ditt profil namn för att börja gälla.
