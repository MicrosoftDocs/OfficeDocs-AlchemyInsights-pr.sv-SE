# <a name="change-your-profile-name"></a>Ändra namn på profilen

1. Gå till [ditt Microsoft-kontoinfo](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. Välj **Redigera namn**under ditt namn.
3. Ange ett förnamn och efternamn och välj **Spara**.

Vänta upp till 24 timmar för ändringar i profilens namn ska börja gälla.
  
Mer information finns i [ändra ditt profilfoto eller namn i Outlook.com](https://go.microsoft.com/fwlink/?linkid=873110).
