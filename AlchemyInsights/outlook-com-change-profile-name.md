# <a name="change-your-profile-name-in-outlookcom"></a>Ändra ditt profilnamn i Outlook.com

1. Gå till [din Microsoft-kontoinformation](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. Välj **Redigera namn**.
3. Ange ett förnamn och efternamn och välj **Spara**.

Det kan ta upp till 24 timmar innan ändringar av ditt profilnamn träder i kraft.
