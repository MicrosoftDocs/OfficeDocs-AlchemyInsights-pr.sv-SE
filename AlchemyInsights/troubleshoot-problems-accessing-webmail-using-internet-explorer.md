# <a name="troubleshoot-problems-with-accessing-webmail"></a>Felsöka problem med åtkomst till webbmeddelanden

Felaktiga datum- och tidsinställningar på datorn kan orsaka problem med åtkomst till webbaserade e-postservrar. Kontrollera att datum- och tidsinställningarna är korrekta genom att följa de här stegen:

1. Välj **Start,** **ange datum och tid** och välj sedan Datum och **tid.**
2. Gör något **& följande** i fönstret Datum och tid:

    - Aktivera Ange **tid automatiskt och** ange tidszon **automatiskt.**

    - Inaktivera automatiska inställningar men välj  knappen Ändra, välj  datum och  tid i dialogrutan Ändra datum och tid och välj sedan **Ändra.** 

Mer information finns i Felsöka [problem med att komma åt webbmeddelanden med hjälp av Internet Explorer.](https://answers.microsoft.com/windows/forum/all/problem-accessing-email-through-ie/41f871f3-6df3-4bc9-a5bd-7f71651a2888)
