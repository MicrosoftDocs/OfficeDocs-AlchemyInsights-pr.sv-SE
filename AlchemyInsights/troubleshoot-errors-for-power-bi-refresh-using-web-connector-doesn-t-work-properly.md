# <a name="refresh-using-web-connector-doesnt-work-properly"></a>Uppdatera med Web Connector fungerar inte korrekt

Om du har ett webb anslutningsskript som använder [Web. Page](https://msdn.microsoft.com/library/mt260924.aspx) -funktionen och du har uppdaterat din datauppsättning eller rapport efter den 18 november 2016, måste du använda en gateway för att uppdateringen ska fungera korrekt.

För mer information:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
