# <a name="find-the-page-that-shows-my-contacts"></a>Hitta den sida som visar mina kontakter

Du kan söka efter dina kontakter på samma sätt som du söker efter dina e-postmeddelanden.
 
Tryck på sökikonen för förstoringsglas och skriv sedan in din kontaktsökbegäran i rutan högst upp på skärmen. **(Protip:** Du kan dubbellyssna på förstoringsglaset för att få fokus till sökrutan.) Dina resultat visas i avsnittet Kontakter.
