# <a name="550-57501-access-denied-spam-abuse-detected"></a>550 5.7.501 åtkomst nekad, skräp post missbruk upptäcktes

Vanligt vis visas det här meddelandet när användare skickar e-postmeddelanden från IP-adresser med *den domän som* är tilldelad till nya klient organisationer i Microsoft 365. Det enklaste sättet att lösa problemet är att:

1. [Lägga till en domän i klient organisationen](https://docs.microsoft.com/microsoft-365/admin/setup/add-domain).

2. [Ändra användarens primära e-postadress](https://docs.microsoft.com/microsoft-365/admin/add-users/change-a-user-name-and-email-address) till den nya anpassade domän du just har lagt till.
