# <a name="creating-public-folders"></a>Skapa gemensamma mappar

Innan användarna kan använda gemensamma mappar måste du skapa minst en postlåda för gemensamma mappar. Så här skapar du:
  
1. Gå till **Administrationscenter för Administrationscenter** \> i Microsoft 365 **.**

2. Navigera till **Delade mappar** \> **Med delade mapppostlådor**och klicka sedan på **Lägg till**.

3. I **Postlåda för delad mapp**anger du ett namn på postlådan för den gemensamma mappen.

4. Klicka på **Spara**.

När en postlåda har skapats kan du börja skapa enskilda mappar eller be användarna att [börja skapa och använda gemensamma mappar i Outlook](https://support.office.com/article/Create-and-share-a-public-folder-in-Outlook-a2835011-d524-4a5c-a207-05c159bb2a97).
