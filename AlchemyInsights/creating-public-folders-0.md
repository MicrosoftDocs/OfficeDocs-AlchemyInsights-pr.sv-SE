# <a name="creating-public-folders"></a>Skapa gemensamma mappar

Innan användarna kan använda gemensamma mappar måste du skapa minst en post låda för gemensam mapp. Så här skapar du:
  
1. Öppna administrations centret för Microsoft 365 och gå **till** \> **Exchange**-tjänsten.

2. Navigera till **gemensamma mappar** i \> **mappen post lådor**och klicka sedan på **Lägg till**.

3. Ange ett namn på post lådan för den offentliga mappen i den **offentliga mappen**.

4. Klicka på **Spara**.

När du har skapat en post låda kan du börja skapa enskilda mappar eller uppmana användarna att [börja skapa och använda gemensamma mappar i Outlook](https://support.office.com/article/Create-and-share-a-public-folder-in-Outlook-a2835011-d524-4a5c-a207-05c159bb2a97).
