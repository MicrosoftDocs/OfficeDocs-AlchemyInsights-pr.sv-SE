# <a name="hybrid-mdm"></a>Hybrid MDM

Microsoft har avslutat hybrid tjänsten för MDM från den 1 september 2019. Eventuella kvarvarande hybrid MDM-enheter tar inte emot principer, appar eller säkerhets uppdateringar. Mer information finns [i vad hände med hybrid MDM?](https://docs.microsoft.com/configmgr/mdm/understand/what-happened-to-hybrid)
