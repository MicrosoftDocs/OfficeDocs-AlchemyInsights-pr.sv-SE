# <a name="public-folder-access-from-mobile-devices"></a>Åtkomst från mobila enheter till gemensam mapp

Gemensamma mappar är inte tillgängliga från mobila enheter.
  
Om du vill ha liknande funktioner som gemensamma mappar och komma åt dem från mobila enheter kan du läsa [Lär dig mer om Microsoft 365-grupper](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) .
