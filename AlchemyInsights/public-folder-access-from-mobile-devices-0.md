# <a name="public-folder-access-from-mobile-devices"></a>Åtkomst till gemensamma mappar från mobila enheter

Gemensamma mappar är inte tillgängliga från mobila enheter.
  
Om du vill ha funktioner som liknar gemensamma mappar och komma åt den från mobila enheter läser [du Lär dig mer om Microsoft 365-grupper](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) för ett bättre alternativ.
