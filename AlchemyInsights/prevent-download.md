# <a name="prevent-files-from-being-downloaded-from-sharepoint-or-onedrive"></a>Förhindra att filer hämtas från SharePoint eller OneDrive

Det finns olika metoder som kan användas för att förhindra eller blockera nedladdningar inträffar. Besök länkarna nedan för respektive metoder:

- [Blockera filhämtning med Azure AD-villkorlig åtkomst](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)

- [Styra åtkomsten från ohanterade enheter](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)

- [Konfigurera IRM-principer för SharePoint-dokumentbibliotek för att begränsa hämtningen av innehåll](https://docs.microsoft.com/office365/securitycompliance/set-up-irm-in-sp-admin-center)
