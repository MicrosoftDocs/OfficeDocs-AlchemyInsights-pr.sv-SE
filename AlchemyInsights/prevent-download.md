# <a name="prevent-files-from-being-downloaded-from-sharepoint-or-onedrive"></a>Förhindra att filer laddas ned från SharePoint eller OneDrive

Det finns olika metoder som kan användas för att förhindra eller blockera nedladdning. Besök länkarna nedan för respektive metod:

- [Blockera fil nedladdning med villkorlig Azure AD-åtkomst](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)

- [Styra åtkomsten från ohanterade enheter](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)

- [Konfigurera IRM-principer på SharePoint-dokumentbibliotek för att begränsa nedladdning av innehåll](https://docs.microsoft.com/microsoft-365/compliance/set-up-irm-in-sp-admin-center)
