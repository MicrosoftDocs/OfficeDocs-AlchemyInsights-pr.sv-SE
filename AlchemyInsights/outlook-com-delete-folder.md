# <a name="delete-a-folder"></a>Ta bort en mapp

1. Högerklicka på den mapp eller undermapp som du vill ta bort och välj **Ta bort mapp**i den vänstra rutan.
2. Välj **OK** för att bekräfta.

Mer information finns i [arbeta med meddelandemappar i Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
