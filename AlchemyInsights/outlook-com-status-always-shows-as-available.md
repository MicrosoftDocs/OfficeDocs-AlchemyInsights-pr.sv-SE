# <a name="my-status-always-shows-as-available"></a>Min status visas alltid som tillgängliga

Varje gång du loggar in på Outlook.com, kommer status för ditt konto att återställas till tillgänglig. Du kan förhindra att din status ändrar inte loggar ut från Outlook.com. Om du vill fortsätta vara inloggad när du stänger webbläsaren, markerar du kryssrutan **Håll mig inloggad** på sidan för Outlook.com.
