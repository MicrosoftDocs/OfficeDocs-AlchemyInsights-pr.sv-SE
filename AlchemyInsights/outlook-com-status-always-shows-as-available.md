# <a name="my-status-always-shows-as-available"></a>Min status visas alltid som Tillgänglig

Varje gång du loggar in på Outlook.com återställs din kontostatus till Tillgänglig. Du kan förhindra att din status ändras genom att inte logga ut från Outlook.com. Om du vill vara inloggad när du stänger webbläsaren väljer du rutan **Håll mig inloggad** på Outlook.com inloggningssidan.
