# <a name="my-status-always-shows-as-available"></a>Min status visas alltid som tillgänglig

Varje gång du loggar in i Outlook.com återställs din konto status till tillgänglig. Du kan förhindra att din status ändras genom att inte logga ut från Outlook.com. Om du vill vara inloggad när du stänger webbläsaren markerar du rutan **Behåll jag inloggad** på inloggnings sidan för Outlook.com.
