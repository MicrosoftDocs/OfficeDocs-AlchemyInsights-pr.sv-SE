# <a name="how-permissions-inheritance-works-in-sharepoint"></a>Så här fungerar behörigheter arv i SharePoint

Som standard ärvs behörigheterna i SharePoint från högre upp i hierarkin. Så ärver en fil dess behörigheter från mappen som har ärvt sina rättigheter från biblioteket som ärver dess behörighet från webbplatsen som ärver dess behörigheter från webbplatssamlingen.
  
Info om ta bort unika behörigheter och återställa arv finns i [Redigera och hantera behörigheter för en lista eller ett bibliotek](https://go.microsoft.com/fwlink/?linkid=869946).
