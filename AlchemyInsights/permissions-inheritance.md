# <a name="how-permissions-inheritance-works-in-sharepoint"></a>Hur behörigheter arv fungerar i SharePoint

Som standard ärvs behörigheterna i SharePoint från högre upp i hierarkin. Så en fil ärver sina behörigheter från mappen, som ärver sina behörigheter från biblioteket, som ärver dess behörigheter från platsen, som ärver sina behörigheter från webbplatssamlingen.
  
Information om hur du tar bort unika behörigheter och återställer arv finns i [Redigera och hantera behörigheter för en lista eller ett bibliotek](https://go.microsoft.com/fwlink/?linkid=869946).
