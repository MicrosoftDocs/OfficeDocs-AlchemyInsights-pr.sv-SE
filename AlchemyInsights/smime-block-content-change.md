# <a name="outlook-will-now-default-block-external-content-in-smime"></a>Outlook kommer nu att blockera externt innehåll i S/MIME

Vi introducerar ytterligare säkerhets åtgärder i Outlook för Windows genom att blockera externt innehåll i S/MIME-krypterade och signerade meddelanden och förhindra åsidosättningar för enskilda meddelanden.

Mer information finns i [Outlook för Windows blockerar externt innehåll i S/MIME-meddelanden](https://support.office.com/article/2d3a4af1-fe41-475f-a888-fc7b997d112e).
