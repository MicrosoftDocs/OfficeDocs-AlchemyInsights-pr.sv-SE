# <a name="create-and-use-a-shared-mailbox"></a>Skapa och använda en delad postlåda

En delad postlåda kan flera personer skicka och ta emot e-post för ett alias, som sales@contoso.com, utan att tilldela ytterligare en licens.
  
- [Gör så här](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) i Microsoft 365 administratörscenter för att komma igång. 

Du kan också läsa artikeln [Skapa en delad postlåda](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9.aspx) , som talar om allt du behöver veta om delade postlådor, inklusive hur du använder dem.
