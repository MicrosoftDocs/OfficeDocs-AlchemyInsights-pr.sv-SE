# <a name="cannot-complete-upgrade-to-teamsonly"></a>Det går inte att slutföra uppgraderingen till TeamsOnly

Användare med ett lokalt Skype för företag-konto kan inte uppgraderas till **TeamsOnly**. Om du har användare som inte har migrerats från lokal kan du inte ändra läget för klient organisation till **TeamsOnly**. När du har flyttat alla användare från lokal till molnet kan du avställa den lokala Skype för företag-distributionen och sedan slutföra klient uppgraderingen till **TeamsOnly**. 

Mer information finns i [inaktivera Hybrid för att slutföra migrering till molnet](https://docs.microsoft.com/skypeforbusiness/hybrid/cloud-consolidation-disabling-hybrid).
