# <a name="find-missing-folders"></a>Söka efter mappar som saknas

Om mapparna inte visas kontrollerar du att mapparna inte är komprimerad. Markera pilen bredvid **mappar** i det vänstra fönstret.
  
Om du tar bort en mapp av misstag kan du återställa den om den finns kvar i mappen Borttaget. Om du vill veta mer kan du läsa [återställa borttagna e-postmeddelanden i Outlook.com](https://support.office.com/article/cf06ab1b-ae0b-418c-a4d9-4e895f83ed50).
  
Det finns ett känt problem där mappar saknas. Vi arbetar för att åtgärda detta. Du kan ändå hitta objekt som sparats i mappar som saknas med hjälp av **sökning**och det kan hjälpa om du loggar ut och sedan logga in igen. [Klicka här om du vill logga ut.](https://login.live.com/logout.srf)
