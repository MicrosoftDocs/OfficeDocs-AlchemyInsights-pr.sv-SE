# <a name="delay-or-schedule-sending-email-messages"></a>Fördröj eller schemalägga e-postmeddelanden

Tyvärr kan du inte skjuta upp eller schemalägga sändningen av e-postmeddelanden i en mobilapp eller webbläsare. För att göra det, Använd [Outlook för Windows eller Outlook för Mac](https://products.office.com/outlook/email-and-calendar-software-microsoft-outlook).

**Ångra skicka**

Du kan ändra dina Outlook.com-inställningar för att fördröja att skicka ett meddelande i upp till 10 sekunder. Du kan avbryta meddelandet under den här perioden. Om du vill aktivera den här funktionen öppnar du inställningar för ångra för att [Skicka](https://outlook.live.com/mail/options/mail/messageContent/undoSend) .

Läs mer om att [ringa upp e-post i Outlook.com](https://support.office.com/article/c069ddde-5282-4085-8f4c-d7b133324f8a?wt.mc_id=Office_Outlook_com_Alchemy).
