# <a name="change-your-language-date-and-time-format-or-time-zone"></a>Ändra språk, datum och tidsformat eller tidszon

1. Överst på sidan, Välj **Inställningar** > **Visa alla Outlook-inställningar**.
2. Välj **Allmänt** > **språk och tid**.
3. Välj det språk du vill använda.
4. Välj **Spara**.

Mer information finns i [Ändra tidszon och språkinställningar i Outlook på webben](https://support.office.com/article/65239869-12e7-4a9d-bca1-76b0ad7ce273).
