# <a name="add-a-user-account-in-windows-10"></a>Lägga till ett användar konto i Windows 10

Så här lägger du till ett konto i Windows 10:

1. Gå till **Start**  >  **Inställningar**  >  **konton**  >  **familje & andra personer** (om **Other people**du använder Windows 10 Enterprise) > **lägga till någon annan på den här datorn**.
2. Följ instruktionerna för att lägga till ett Microsoft-konto eller ett lokalt konto.

[Lär dig mer om hur du konfigurerar konton](https://support.microsoft.com/help/17197/)
