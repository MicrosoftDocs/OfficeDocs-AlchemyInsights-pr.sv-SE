# <a name="enable-versioning-for-a-list-or-library"></a>Aktivera versions hantering för en lista eller ett bibliotek

1. Gå till listan eller biblioteket.
    
2. Klicka på **Inställningar**och sedan på **biblioteks inställningar** eller **list inställningar**.
    
3. Klicka på **versions inställningar**.
    
4. Välj önskade inställningar och klicka sedan på **OK**.
