# <a name="enable-versioning-for-a-list-or-library"></a>Aktivera versionshantering för en lista eller ett bibliotek

1. Gå till listan eller biblioteket.
    
2. Klicka på **Inställningar**och klicka sedan på **Inställningar för dokumentbibliotek** eller **listinställningar**.
    
3. Klicka på **Inställningar för versionshantering**.
    
4. Välj önskade inställningar och klicka sedan på **OK**.
