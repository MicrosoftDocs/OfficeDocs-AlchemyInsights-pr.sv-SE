# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>550 5.7.705 åtkomst nekades, klient organisationen har överskridit tröskelvärdet

Det här meddelandet visas när du har skickat för mycket skräp post eller Mass utskick av din organisation och gjort ett block på utgående e-post.
Vanliga orsaker är problem med lokala servrar eller administratörs konton som har använts för att skapa kopplingar. Ett av villkoren kan tillåta att skräp post passerar genom din organisation.

Om du vill ta bort det här blocket måste du förstå och förklara orsaken till en support agent samt åtgärda det underliggande problemet.
Klicka på **kontakta oss** nedan när du är klar.
