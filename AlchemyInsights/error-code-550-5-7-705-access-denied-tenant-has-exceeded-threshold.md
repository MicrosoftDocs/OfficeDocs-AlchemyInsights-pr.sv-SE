# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>550 5.7.705 Nekad åtkomst, klienten har överskridit tröskelvärdet

Det här meddelandet inträffar när för mycket skräppost eller massutskick har skickats av din organisation och vi placerar ett block på utgående e-post.
Vanliga orsaker är komprometterade lokala servrar eller komprometterade administratörskonton som har använts för att skapa kopplingar. Båda villkoren kan tillåta spam att passera genom din organisation.

Om du vill ta bort det här blocket måste du förstå och förklara orsaken till en supportagent samt korrigera det underliggande problemet.
Klicka på **Kontakta oss** nedan när du är redo.
