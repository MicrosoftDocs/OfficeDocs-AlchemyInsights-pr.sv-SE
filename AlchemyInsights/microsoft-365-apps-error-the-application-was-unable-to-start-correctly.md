# <a name="microsoft-365-apps-error-the-application-was-unable-to-start-correctly-0xc0000142"></a>Microsoft 365-program fel: programmet kunde inte startas korrekt (0xc0000142)

För att åtgärda det här felet kan du försöka med följande:

- Utför en online-reparation av Office och starta om datorn. Mer information finns i [reparera ett Office-program](https://support.microsoft.com/office/repair-an-office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b).
- Gå till **Start**   >   **Kör**och skriv **Services. msc**. Högerklicka på  **Microsoft Office Klicka-och-kör-tjänsten**och välj sedan **starta om**.
- Reparera saknade eller skadade systemfiler med system fils granskaren och starta om datorn. Mer information finns i [använda system fils granskaren för att reparera saknade eller skadade systemfiler](https://support.microsoft.com/help/929833/use-the-system-file-checker-tool-to-repair-missing-or-corrupted-system).
- Avinstallera och installera om Office. Mer information finns i [Avinstallera Office från en dator](https://support.microsoft.com/office/uninstall-office-from-a-pc-9dd49b83-264a-477a-8fcc-2fdf5dbf61d8).

Mer information finns i:  

- [Fel vid öppning av Microsoft Office-program](https://support.office.com/article/error-when-opening-microsoft-office-apps-b84b6a63-4b8c-46ec-ae9a-ad91d6160d72)  

- [Jag får ett fel meddelande om att det slutat fungera när jag startar Office-program på min dator](https://support.office.com/article/i-get-a-stopped-working-error-when-i-start-office-applications-on-my-pc-52bd7985-4e99-4a35-84c8-2d9b8301a2fa)
