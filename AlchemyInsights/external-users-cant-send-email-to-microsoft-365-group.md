# <a name="external-users-cant-send-email-to-microsoft-365-group"></a>Externa användare kan inte skicka e-post till Microsoft 365-gruppen

Så här gör du så att externa användare kan skicka e-post till din Microsoft 365-grupp:

1. Logga in på grupper i [administrationscentergrupper](https://admin.microsoft.com/)  >  **Groups**  >  **Groups**.
2. Välj den grupp som du vill hantera för att öppna fönstret Inställningar.
3. På fliken **Inställningar** väljer du **Tillåt externa avsändare att skicka den här gruppen till e-post**.
4. Välj **Spara**.</br>
    Tillåt upp till en timme innan inställningarna ska visas.
