# <a name="form-header---web-resource"></a>Formulärhuvud - webbresurs

Du kan lägga till eller redigera webbresurser i ett formulär för att göra formuläret mer tilltalande eller användbart för användarna. Börja med att kontrollera att du har säkerhetsrollen Systemadministratör eller Systemanpassare, eller motsvarande, behörigheter.  

**Anm.)** Du kan inte inkludera en webbresurs i ett formulärhuvud eller en sidfot.

Mer information finns i [Skapa och redigera en webbresurs i ett formulär](https://docs.microsoft.com/dynamics365/customer-engagement/customize/create-edit-web-resources#create-and-edit-a-web-resource-on-a-form).
