# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>Installera, konfigurera och synkronisera OneDrive-eller SharePoint-filer på Mac 

Installera, konfigurera och synkronisera OneDrive-eller SharePoint-filer på en Mac med instruktionerna från [Synkronisera filer med OneDrive-synkroniseringsklienten på Mac OS X](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f).

Se [Åtgärda synkroniseringsproblem med OneDrive (Mac)](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756)om du har installerat och konfigurerat OneDrive-programmet på en Mac-klient.

> [!NOTE]
> Funktionen för att flytta mappen kända filer i OneDrive-synkronisering för Windows är inte tillgänglig för Mac.
