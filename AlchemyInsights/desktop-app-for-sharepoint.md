# <a name="desktop-app-for-sharepoint"></a>Desktop app för SharePoint

Din organisation anges i privata Preview-programmet för Desktop-App för SharePoint. Microsoft stöder för närvarande inte Desktop-App för SharePoint (privat förhandsgranskning). Använd Microsoft privata Förhandsgranska medverkande partner för att få hjälp med Desktop-App för SharePoint.
