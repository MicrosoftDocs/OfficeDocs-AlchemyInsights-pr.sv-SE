# <a name="desktop-app-for-sharepoint"></a>Skriv bords program för SharePoint

Din organisation anges i det privata för hands versions programmet för Skriv bords programmet för SharePoint. Microsoft stöder för närvarande inte Skriv bords programmet för SharePoint (privat för hands version). Använd dina samanvända Microsoft-partners för privat för hands version för att få hjälp med Skriv bords programmet för SharePoint.
