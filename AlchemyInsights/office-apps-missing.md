# <a name="microsoft-365-apps-missing"></a>Microsoft 365-appar saknas

Om du inte kan hitta Office-program på datorn efter en installation eller uppdatering [läser du Kan inte hitta Office-program i Windows 10, Windows 8 eller Windows 7?](https://support.office.com/article/Can-t-find-Office-applications-in-Windows-10-Windows-8-or-Windows-7-907ce545-6ae8-459b-8d9d-de6764a635d6) Om du inte hittar Office-genvägarna på Start-menyn eller under **Alla appar**läser du [Microsoft 365-appar som saknas i alla appar i Windows 10](https://support.office.com/article/office-apps-are-missing-from-all-apps-on-windows-10-5bc123f6-655d-4736-ad61-b0b9d1cde5bc). 

Försök att [reparera ett Office-program](https://support.office.com/article/repair-an-office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b) om Office-genvägar försvann efter att du loggat in eller startat om datorn. 

**Anm.:** Om Microsoft 365-appar saknas för installation på Office-portalen går du till [Felsöka installation av Office](https://support.office.com/article/troubleshoot-installing-office-35ff2def-e0b2-4dac-9784-4cf212c1f6c2) och gå till Vanliga problem **för Office för företag**Du  >  **Common issues**  >  **ser inget alternativ för att installera Office**.
