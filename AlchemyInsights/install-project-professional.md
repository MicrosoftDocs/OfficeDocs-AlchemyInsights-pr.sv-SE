Se [installera Project](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) för att lära dig hur du installerar abonnemangs versionen av Project Online och den icke-prenumerations versionen av Project 2019, 2016 eller 2013. 

För Project 2010 eller 2007 följer du stegen i [office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d)   eller [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

Project säljs som ett fristående program, så om du inte har den kan du [köpa eller prova Project](https://products.office.com/project).
