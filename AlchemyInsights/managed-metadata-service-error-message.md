# <a name="managed-metadata-service-error-message"></a>Felmeddelande om tjänsten Hanterade metadata

När du öppnar termlagringsplatsen kan du se ett felmeddelande: "Tjänsten hanterade metadata eller anslutningen är inte tillgänglig för närvarande. Webbtjänsten Programpool eller Hanterade metadata kanske inte har startats. Kontakta administratören."
  
Om det har gått mer än 24 timmar sedan du först såg det här problemet loggar du ett supportärende. Vi förstår att 24 timmar kan verka som en lång tid. I många fall arbetar vi redan på en lösning så snälla ge oss minst 24 timmar.
