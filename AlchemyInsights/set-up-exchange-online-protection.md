# <a name="set-up-exchange-online-protection"></a>Aktivera Exchange Online Protection

Microsoft Exchange Online Protection (EOP) är en molnbaserad tjänst för e-postfiltrering för skydd mot skräp post och skadlig program vara, med funktioner för att skydda din organisation från överträdelser av meddelande principer.

Använd [konfigurations guiden för Exchange Online-skydd](https://admin.microsoft.com/adminportal/home#/modernonboarding/prepareyourenvironment) för att konfigurera EOP genom att välja bland de tre distributions scenarierna, hybrid post lådor (blandning av lokala och moln) post lådor eller alla moln post lådor – som passar din organisation. Guiden innehåller information och resurser som du kan använda för att konfigurera EOP och granska användarens licensiering, tilldela behörigheter i administrations centret för Microsoft 365 och konfigurera organisationens & principer mot skadlig program vara och skräp post.
