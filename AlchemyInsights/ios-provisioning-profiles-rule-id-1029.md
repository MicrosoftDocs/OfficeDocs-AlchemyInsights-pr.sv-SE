# <a name="ios-provisioning-profiles"></a>iOS-etablerings profiler

I Apples iOS-line-Business används begreppet etablerings profil som används för att verifiera programmets integritet och påtvinga principer som definierats i profilen. Dessa Provisioning-profiler upphör efter 12 månader och därför måste en ny profil vara associerad med den distribuerade appen för att programmen ska kunna köras.
  
I följande artikel beskrivs hur du skapar en ny konfigurations profil med en iOS-LOB-app distribuerad genom Intune: [Använd Provisioning-profilerna för iOS-appen för att förhindra att dina appar upphör att gälla](https://docs.microsoft.com/intune/app-provisioning-profile-ios)
