# <a name="turn-off-focused-inbox-in-email-clients"></a>Inaktivera fokuserad inkorg i e-postklienter

Om du vill stänga av fokuserad inkorg i Outlook 2016, Outlook på webben, Outlook.com/Hotmail.com, Outlook 2016 för Mac eller mail för Windows 10, se [stänga av fokuserad inkorg](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2.aspx).

Om du vill inaktivera fokuserad inkorg för alla i organisationen, se [Konfigurera fokuserad inkorg för alla i](https://docs.microsoft.com/office365/admin/setup/configure-focused-inbox)organisationen.
