# <a name="problems-opening-office-files-from-sharepoint"></a>Problem med att öppna Office-filer från SharePoint

Om du har problem med att öppna Office-filer och dokument från SharePoint på lokala Office-klienter läser du länkarna nedan. 

- [Kontrollera fil begränsningar för att säkerställa att filen stöds](https://support.office.com/article/Invalid-file-names-and-file-types-in-OneDrive-OneDrive-for-Business-and-SharePoint-64883a5d-228e-48f5-b3d2-eb39e07630fa).

- [Ta bort Office-dokumentcachen](https://support.office.com/article/Delete-your-Office-Document-Cache-b1d3765e-d71b-4bb8-99ca-acd22c42995d).

- [Kör en reparation på kontoret](https://support.office.com/Article/Repair-an-Office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b).

För mer information eller Detaljer hänvisas till artikeln, [åtgärda problem med att öppna dokument i SharePoint-bibliotek](https://support.office.com/article/Fix-problems-opening-documents-in-SharePoint-libraries-31329FA1-4AD0-47FC-95D8-BB0C5B12A536).

**Anmärkning:** Om du har problem med dokument som öppnas i webbläsaren eller klienten som standard kan du behöva ändra biblioteks inställningen. För att göra detta, navigera till biblioteket har problem > redskap ikonen i övre högra hörnet > **biblioteksinställningar** >  **Avancerade inställningar** > **öppna dokument i webbläsaren**.
