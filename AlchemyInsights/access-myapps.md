# <a name="access-myapps"></a>Komma åt Mina appar

Om du har behörighet att komma åt och använder organisationens molnbaserade program kan du nå dem via portalen mina program. Mer information finns i [Access och använda portalen mina program på datorn](https://docs.microsoft.com/azure/active-directory/user-help/my-apps-portal-end-user-access#access-and-use-the-my-apps-portal-on-your-computer).

Du kan också komma åt program via samlingarna. Mer information finns i [Access-program med hjälp av samlingar](https://docs.microsoft.com/azure/active-directory/user-help/my-applications-portal-workspaces#access-apps-using-collections).

Om du har problem med att logga in på portalen mina program kan du läsa det [går inte att logga in på portalen mina program](https://docs.microsoft.com/azure/active-directory/user-help/my-apps-portal-end-user-troubleshoot#i-cant-sign-in-to-the-my-apps-portal) .

Om du får ett meddelande om **nekad åtkomst** efter att du har startat ett program från My app-portalen kan [jag se att jag får ett meddelande om nekad åtkomst när jag startar en app](https://docs.microsoft.com/azure/active-directory/user-help/my-apps-portal-end-user-troubleshoot#im-getting-an-access-denied-message-when-i-start-an-app).
