# <a name="sendas-mail-enabled-public-folder"></a>Skicka som e-postaktiverad gemensam mapp

I följande exempel tilldelas "Send As"-behörigheter för den e-postaktiverade gemensamma mappen NewPF1 till användarens Jason.

Add-RecipientPermission-Identity ' NewPF1 '-förvaltaren "Jason"-AccessRights ' sende '

Detaljerad information om syntax och parametrar finns i [tilldela behörigheter för att skicka som eller skicka för e-postaktiverade gemensamma mappar](https://docs.microsoft.com/exchange/collaboration-exo/public-folders/assign-permissions-mail-enabled-pfs).
