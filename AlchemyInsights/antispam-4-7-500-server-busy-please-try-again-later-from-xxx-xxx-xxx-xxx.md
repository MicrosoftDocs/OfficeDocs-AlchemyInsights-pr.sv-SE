# <a name="47500-server-busy-please-try-again-later"></a>4.7.500 Server upptagen, försök igen senare

Det här felet uppstår när volymen av e-posttrafik från källans IP-adress överskrider gränsen baserat på rykte (eller brist på rykte) för källans IP-adress.

Blockering av e-post från källans IP-adress upphör att gälla inom en timme. Om käll-IP-adressen är en lokal e-postserver som tillhör dig kontrollerar du konfigurationen av e-postflödeskopplingen. Om beteendet fortsätter i mer än en timme kontaktar du supporten för att begära ett undantag för källans IP-adress.
