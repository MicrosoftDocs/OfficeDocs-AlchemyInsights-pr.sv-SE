# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Kan du få ett konto inaktiverat fel i Outlook på webben?

Om du får felet Ditt konto `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` har **inaktiverats** när du försöker öppna Outlook på webben (tidigare kallat Outlook Web App) kan din organisationsadministratör ha inaktiverat din åtkomst till Outlook på webben. Administratören kan åtgärda det här felet genom att följa stegen i avsnittet [Aktivera eller inaktivera Outlook Web App för en postlåda](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
