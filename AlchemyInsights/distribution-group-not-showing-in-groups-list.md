# <a name="distribution-group-not-showing-in-groups-list"></a>Distributionsgrupp som inte visas i listan grupper

Det kan ta upp till 60 minuter innan den nya distributionslistan visas i administratörscenter.
  
Om du vill lägga till medlemmar eller göra andra ändringar just nu använder du funktionen Gruppera i [Exchange Admin Center](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1).
