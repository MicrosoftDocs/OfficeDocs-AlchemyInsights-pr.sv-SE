# <a name="sync-a-sharepoint-library-with-onedrive-sync"></a>Synkronisera ett SharePoint-bibliotek med OneDrive-synkronisering

Om du vill att Dina SharePoint-filer ska synkroniseras med den lokala hårddisken gör du följande:

1. Navigera till det dokumentbibliotek som du vill synkronisera med den lokala hårddisken.
2. Välj knappen Synkronisera på menyalternativen.
3. Följ steg för steg för att slutföra synkroniseringen.
