# <a name="unable-to-attach-or-upload-files-from-facebook"></a>Det går inte att bifoga eller ladda upp filer från Facebook

På grund av ändrade villkor för Facebook, efter den 23 oktober 2019, kommer du inte längre att kunna ladda upp bilagor direkt från Facebook, och Facebook visas inte längre i listan över dina molnplatser. 

Du kan hantera din lista över andra molnplatser i dina [lagringskonton](https://go.microsoft.com/fwlink/?linkid=2111075).

Läs mer om [Lagringskonton](https://support.office.com/article/477cb7cc-5732-4c40-8f23-30472de8138a).
