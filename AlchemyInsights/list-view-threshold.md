# <a name="list-view-threshold"></a>Tröskelvärde för listvy

Tröskelvärdet för listvyn finns på plats för att upprätthålla tjänstens prestanda. Tröskelvärdet kan inte ändras i SharePoint Online. Undvik att nå det här tröskelvärdet [i Hantera stora listor och bibliotek i SharePoint](https://support.office.com/article/manage-large-lists-and-libraries-in-sharepoint-b8588dae-9387-48c2-9248-c24122f07c59).

Du kan också få det här felmeddelandet när du försöker dela eller bryta arvet för en mapp. Se [Feldelning/brytarv](https://docs.microsoft.com/SharePoint/troubleshoot/lists-and-libraries/error-share-break-inheritance).
