# <a name="learn-about-microsoft-365-groups"></a>Lär dig mer om Microsoft 365-grupper

Microsoft 365-grupper är en tjänst som fungerar med de Office 365-verktyg du använder redan, så du kan samar beta med dina grupp medlemmar när du skriver dokument, skapar kalkyl blad, arbetar med projektplaner, schemalägger möten eller skickar e-post. Mer information finns i [Lär dig mer om Microsoft 365 Groups](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
