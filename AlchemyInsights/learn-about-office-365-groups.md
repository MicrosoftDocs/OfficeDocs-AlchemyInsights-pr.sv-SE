# <a name="learn-about-microsoft-365-groups"></a>Läs mer om Microsoft 365-grupper

Microsoft 365 Groups är en tjänst som redan fungerar med de Office 365-verktyg som du redan använder, så att du kan samarbeta med dina lagkamrater när du skriver dokument, skapar kalkylblad, arbetar med projektplaner, schemalägger möten eller skickar e-post. Mer information finns i [Lär dig mer om Microsoft 365 Groups](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
