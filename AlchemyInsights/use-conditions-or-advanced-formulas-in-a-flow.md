# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>Använda villkor och avancerade formler i ett flöde

I Microsoft Flow kan du anpassa ett flöde så att det endast utför uppgifter om ett villkor är sant. Mer information finns i [lägga till ett villkor i ett flöde](https://go.microsoft.com/fwlink/?linkid=872112).
  
Information om hur du skapar en avancerad formel finns i avsnittet [Workflow Definition Language schema för Azure Logic-appar](https://aka.ms/logicexpressions).
