# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>Använda villkor eller avancerade formler i ett flöde

I Microsoft Flow kan du anpassa ett flöde så att det bara utför uppgifter om ett villkor är sant. Information finns i [Lägga till ett villkor i ett flöde](https://go.microsoft.com/fwlink/?linkid=872112).
  
Information om hur du skapar en avancerad formel finns i [Språkschema för arbetsflödesdefinition för Azure-logikappar](https://aka.ms/logicexpressions).
