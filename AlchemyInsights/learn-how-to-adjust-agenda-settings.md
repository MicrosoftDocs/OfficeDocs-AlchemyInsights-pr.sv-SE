# <a name="learn-how-to-configure-agenda-settings"></a>Lär dig mer om hur du konfigurerar inställningar för agenda

Agenda-inställningarna finns i inställningar för **Kalender** i Outlook på webben. Mer information finns i [Inställningar för kalender i Outlook på webben](https://support.office.com/article/12cba5a4-4f95-4d00-bfc3-b694aa67ac8f).

Administratörer kan använda cmdlet **Set-CalendarNotification** i Exchange Online PowerShell för att konfigurera inställningar för agenda. Mer information finns i [Ange CalendarNotification](https://technet.microsoft.com/library/dd351284).
