# <a name="learn-how-to-configure-agenda-settings"></a>Läs om hur du konfigurerar inställningar för dagordning

Agenda-inställningar finns i **kalenderinställningarna** i Outlook på webben. Mer information finns [i Kalenderinställningar i Outlook på webben](https://support.office.com/article/12cba5a4-4f95-4d00-bfc3-b694aa67ac8f).

Administratörer kan använda cmdleten **Set-CalendarNotification** i Exchange Online PowerShell för att konfigurera dagordningsinställningar. Mer information finns i [Ange kalenderanmälelse](https://technet.microsoft.com/library/dd351284).
