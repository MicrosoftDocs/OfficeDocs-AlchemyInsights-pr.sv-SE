# <a name="skype-for-business-common-issues-and-resolutions"></a>Vanliga problem och lösningar för Skype för företag 

Om du har frågor om Skype för företag kan du omformulera frågan för att beskriva problemet, inklusive fel, funktioner som du använder osv. 

Här är några vanliga ämnen:

**Felsöka inloggnings problem i Skype för företag:** Kontrol lera först att [Skype för företag-tjänsten](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/servicehealth) är felfri. Om det är det kan du gå igenom [fel söknings felen i Skype för företag – Online](https://docs.microsoft.com/SkypeForBusiness/set-up-skype-for-business-online/troubleshooting-sign-in-errors-for-admins#check-for-common-causes-of-skype-for-business-online-sign-in-errors) för att eliminera vanliga orsaker till inloggnings problem.
 
**Felsöka ljud och video i Skype för företag:** Granska [lösningarna på vanliga problem med ljud och video i Skype för företag](https://support.office.com/article/Troubleshoot-audio-and-video-in-Skype-for-Business-62777bc6-c52b-47ae-84ba-a8905c3b71dc). 

**Felsöka anslutnings problem i Skype för företag:** Granska [lösningarna på vanliga anslutnings problem i Skype för företag](https://support.office.com/article/troubleshoot-connection-issues-in-skype-for-business-ca302828-783f-425c-bbe2-356348583771).

**Kommunicera med externa Skype för företag-kontakter (Federation):** [Konfigurera Skype för företag för att kontakta externa Skype för företag-användare](https://docs.microsoft.com/SkypeForBusiness/set-up-skype-for-business-online/allow-users-to-contact-external-skype-for-business-users).

**Kommunicera med Skype-kontakter:** [Låt Skype för företag-användare lägga till Skype-kontakter](https://docs.microsoft.com/SkypeForBusiness/set-up-skype-for-business-online/let-skype-for-business-users-add-skype-contacts).

**Uppgradera Skype för företag till Microsoft Teams:** Granska [komma igång med din Microsoft Teams-uppgradering](https://docs.microsoft.com/microsoftteams/upgrade-start-here) och lär dig mer om [samexistens](https://docs.microsoft.com/microsoftteams/coexistence-chat-calls-presence) med Microsoft Teams när du deltar i din organisation.
