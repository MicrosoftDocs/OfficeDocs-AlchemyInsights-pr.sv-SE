# <a name="skype-for-business-common-issues-and-resolutions"></a>Vanliga problem och lösningar för Skype för företag 

För specifika frågor om Skype för företag, kortfattat omformulera din fråga för att beskriva ditt problem, inklusive fel, funktioner du använder, etc. 

Här är några ofta efterfrågade ämnen:

**Felsöka Skype för Business-inloggning:** Kontrollera först att [Skype för Business-tjänsten](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/servicehealth) är felfri. Om det är, granska [Felsökning av Skype för Business Online-Inloggningsfel](https://docs.microsoft.com/SkypeForBusiness/set-up-skype-for-business-online/troubleshooting-sign-in-errors-for-admins#check-for-common-causes-of-skype-for-business-online-sign-in-errors) för att eliminera vanliga orsaker till inloggningsproblem.
 
**Felsöka ljud och video i Skype för företag:** Granska [lösningarna på vanliga ljud-och videoproblem i Skype för företag](https://support.office.com/article/Troubleshoot-audio-and-video-in-Skype-for-Business-62777bc6-c52b-47ae-84ba-a8905c3b71dc). 

**Felsöka anslutningsproblem i Skype för företag:** Granska [lösningarna på vanliga anslutningsproblem i Skype för företag](https://support.office.com/article/troubleshoot-connection-issues-in-skype-for-business-ca302828-783f-425c-bbe2-356348583771).

**Kommunicera med externa Skype för Business-kontakter (Federation):** [Konfigurera Skype för företag att kontakta externa Skype för företag-användare](https://docs.microsoft.com/SkypeForBusiness/set-up-skype-for-business-online/allow-users-to-contact-external-skype-for-business-users).

**Kommunicera med Skype-kontakter:** [Låt Skype för företag-användare lägga till Skype-kontakter](https://docs.microsoft.com/SkypeForBusiness/set-up-skype-for-business-online/let-skype-for-business-users-add-skype-contacts).

**Uppgradera Skype för företag till Microsoft Teams:** Granska [komma igång med din Microsoft Teams-uppgradering](https://docs.microsoft.com/en-us/microsoftteams/upgrade-start-here) och lär dig mer om [samexistens](https://docs.microsoft.com/microsoftteams/coexistence-chat-calls-presence) med Microsoft Teams när du antar team i din organisation.
