# <a name="pdf-reader-in-microsoft-edge"></a>PDF-läsare i Microsoft Edge

PDF-filernas popularitet är ett fokus på behovet av en tillförlitlig, säker och kraftfull PDF-läsare som kan antas av företags företag. Den funktions rika PDF-läsaren som är inbyggd i Microsoft Edge (version 77 och senare) är bara ett sådant program. Den fungerar i Windows och macOS och kan användas för att öppna och kommentera PDF-filer som är lokala, online eller inbäddade på webb sidor.

Mer information finns i [PDF-läsare i Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2140005).
