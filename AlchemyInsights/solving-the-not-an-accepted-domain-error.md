# <a name="got-a-parameter-cannot-be-found-error"></a>Har du fel meddelandet "det går inte att hitta en parameter"?

Om du får fel meddelandet **"det går inte att hitta en parameter som matchar parameter namnet" postadresser ' "** innebär det att det tar lite längre tid för Microsoft 365 att slutföra konfigurationen av klient organisationen eller din egen domän om du nyligen har lagt till en.
  
Konfigurationen kan ta upp till 4 timmar. Vänta en stund så att konfigurationen hinner slutföras och försök sedan igen.
