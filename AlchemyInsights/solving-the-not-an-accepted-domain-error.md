# <a name="got-a-parameter-cannot-be-found-error"></a>Har du ett fel i parametern?

Om felmeddelandet **"En parameter inte kan hittas som matchar parameternamnet "EmailAddresses"** betyder det att det tar lite längre tid för Microsoft 365 att slutföra inrättandet av din klient, eller din anpassade domän om du nyligen lagt till en.
  
Konfigurationen kan ta upp till 4 timmar. Vänta en stund så att konfigurationen hinner slutföras och försök sedan igen.
