# <a name="block-file-download-with-azure-ad-conditional-access"></a>Blockera filhämtning med Azure AD-villkorlig åtkomst

Med cloud app-säkerhet-sessionsprinciper kan du begränsa en session baserat på enhetstillstånd. Om du vill utföra kontrollen över en session med enheten som ett villkor skapar du både en princip för villkorlig åtkomst och en sessionsprincip.

- [Blockera filhämtning med Azure AD-villkorlig åtkomst](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)
