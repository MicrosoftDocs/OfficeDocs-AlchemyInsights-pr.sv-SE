# <a name="screen-sharing-with-skype-consumer-users"></a>Skärmdelning med Skype-konsumentanvändare

Allmänna instruktioner om hur du delar ditt program, om du vill veta mer om begränsningar med delning och för felsökning av resurser, granska [dela din skärm i Skype för företag](https://support.office.com/article/Share-your-screen-in-Skype-for-Business-2d436dc9-d092-4ef1-83f1-dd9f7a7cd3fc).  

För närvarande kan Skype för företag-användare inte skärmdelning med Skype-konsumentanvändare. [Klicka här](https://www.skypefeedback.com/forums/299913-generally-available/suggestions/12335259-enable-screen-sharing-to-consumer-skype-users) för att ge feedback om hur du aktiverar skärmdelning med konsumenternas Skype-användare.
