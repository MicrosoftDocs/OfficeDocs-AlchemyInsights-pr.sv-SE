# <a name="screen-sharing-with-skype-consumer-users"></a>Skärm delning med Skype konsument användare

Om du vill veta mer om begränsningar med delning och om hur du felsöker resurser kan du läsa om hur du delar dina program [i Skype för företag](https://support.office.com/article/Share-your-screen-in-Skype-for-Business-2d436dc9-d092-4ef1-83f1-dd9f7a7cd3fc).  

För närvarande går det inte att dela Skype för företag-användare med Skype-användare. [Klicka här](https://www.skypefeedback.com/forums/299913-generally-available/suggestions/12335259-enable-screen-sharing-to-consumer-skype-users) för att ge feedback om att aktivera skärm delning med Skype-användare.
