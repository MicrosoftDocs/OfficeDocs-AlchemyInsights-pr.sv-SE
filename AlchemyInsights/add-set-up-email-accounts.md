# <a name="problems-setting-up-an-exchange-email-account-in-outlook"></a>Problem med att konfigurera ett Exchange-e-postkonto i Outlook

Om du har problem med att konfigurera ett Exchange-konto i Outlook, kan du köra [installationsverktyget för Outlook-konto](https://aka.ms/SaRA-OutlookSetupProfile) på datorn. Den utför olika kontroller och ger lösningar till ditt konto installationsproblem.
  
Om du fortfarande har problem, finns i [Outlook inte kan konfigurera en ny profil med hjälp av Exchange Autodiscover för en Exchange Online-postlåda i Office 365](https://support.microsoft.com/help/2404385/outlook-can-t-set-up-a-new-profile-by-using-exchange-autodiscover-for).
