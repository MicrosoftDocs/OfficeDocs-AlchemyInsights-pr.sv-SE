# <a name="problems-setting-up-an-email-account-in-outlook"></a>Problem med att konfigurera ett e-postkonto i Outlook

Om du har problem med att konfigurera ett Exchange-konto i Outlook kan du köra [installations verktyget för Outlook](https://aka.ms/SaRA-OutlookSetupProfile) på datorn. Det utför olika kontroller och ger eventuella lösningar på ditt konto konfigurations problem.
  
Om du fortfarande har problem kan du läsa [Outlook kan inte konfigurera en ny profil med hjälp av Autodiscover för Exchange Online](https://docs.microsoft.com/exchange/troubleshoot/outlook-profiles/cannot-set-up-profile-autodiscover).
