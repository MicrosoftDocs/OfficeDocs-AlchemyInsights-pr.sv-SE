# <a name="problems-setting-up-an-exchange-email-account-in-outlook"></a>Problem med att konfigurera ett Exchange-e-postkonto i Outlook

Om du har problem med att konfigurera ett Exchange-konto i Outlook kan du köra [verktyget Outlook Account Setup](https://aka.ms/SaRA-OutlookSetupProfile) på den aktuella datorn. Det kommer att utföra olika kontroller och ge möjliga lösningar på ditt konto installationsproblem.
  
Om du fortfarande har problem, se [Outlook kan inte ställa in en ny profil med hjälp av Exchange Autodiscover för en Exchange Online-postlåda i Office 365](https://docs.microsoft.com/exchange/troubleshoot/outlook-profiles/cannot-set-up-profile-autodiscover).
