# <a name="sharing-advanced-hunting-queries"></a>Dela avancerade frågor

Avancerade jakt frågor kan delas mellan användare i samma organisation. Du kan också hitta frågor som delas offentligt på GitHub. De här frågorna gör det möjligt för dig att snabbt hitta speciella hot om hotet utan att behöva skriva frågor från grunden.
  
Mer information finns i [använda delade frågor i Advanced jakt](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/advanced-hunting-shared-queries).
