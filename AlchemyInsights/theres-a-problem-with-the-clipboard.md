# <a name="resolving-excel-clipboard-error"></a>Åtgärda fel i Excel

Om du får ett fel meddelande i Excel med texten "det finns ett problem med Urklipp, men du kan fortfarande klistra in innehållet i den här arbets boken" prova att inaktivera funktionen för dynamisk förhands granskning genom att följa de här stegen:

1. Klicka på **fil**  >  **alternativ**i Excel.
3. Välj **Allmänt**i dialog rutan **Excel-alternativ** .
4. Avmarkera **aktivera dynamisk förhands granskning**under **alternativ för användar gränssnitt**.
5. Klicka på **OK**.
6. Starta om Excel.
