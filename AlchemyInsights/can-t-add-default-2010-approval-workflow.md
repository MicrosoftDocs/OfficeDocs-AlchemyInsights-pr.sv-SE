# <a name="cant-add-default-2010-approval-workflow"></a>Det går inte att lägga till arbets flödet standard 2010-godkännande

I en Microsoft SharePoint-webbplatssamling kan du inte lägga till ett globalt återanvändbart arbets flöde (som "godkännande-SharePoint 2010") i en lista eller ett bibliotek. Godkännande arbets flödet kanske saknas. Du kan lösa problemet genom att [lägga till ett globalt återanvändbart arbets flöde i en SharePoint-lista eller ett bibliotek](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library).

Mer information om arbets flödes godkännanden finns i [allt om arbets flöden för godkännande](https://support.office.com/article/All-about-Approval-workflows-078C5A89-821F-44A9-9530-40BB34F9F742). 
 
Läs mer om [moderna godkännanden i flödet](https://flow.microsoft.com/blog/introducing-modern-approvals).
