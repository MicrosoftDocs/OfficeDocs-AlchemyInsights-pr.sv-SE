# <a name="cant-add-default-2010-approval-workflow"></a>Det går inte att lägga till standard 2010 arbetsflöde för godkännande

Du kan inte lägga till ett globalt återanvändbart arbetsflöde (till exempel ”godkännande - SharePoint 2010”) i en Microsoft SharePoint-webbplatssamling till en lista eller ett bibliotek. Arbetsflöde för godkännande kan saknas. Lös problemet genom att hänvisa till [kan inte lägga till ett globalt återanvändbart arbetsflöde till en SharePoint-lista eller ett bibliotek](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library).

Mer information om arbetsflödesgodkännanden finns [alla arbetsgångar för godkännande](https://support.office.com/article/All-about-Approval-workflows-078C5A89-821F-44A9-9530-40BB34F9F742). 
 
Läs mer om [Moderna godkännanden i flödet](https://flow.microsoft.com/blog/introducing-modern-approvals).
