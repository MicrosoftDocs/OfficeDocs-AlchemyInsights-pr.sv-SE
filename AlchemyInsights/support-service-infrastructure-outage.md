# <a name="support-service-infrastructure-outage"></a>Stöd för tjänst infrastruktur

Vi håller på att drabbas av support tjänstens infrastruktur. Vi arbetar på att lösa problemet så fort som möjligt, men under tiden kan du se svars tids fördröjningar när du försöker kontakta Microsoft.
