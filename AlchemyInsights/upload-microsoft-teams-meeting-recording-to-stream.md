# <a name="upload-a-microsoft-teams-meeting-recording-to-stream-a1"></a>Ladda upp en inspelning med Microsoft Teams-möte till ström (a1)

De flesta Microsoft 365-användare laddas upp automatiskt till Microsoft Stream efter ett möte. Från och med början den  **20 augusti 2020**, när användare med a1 (inklusive a1 plus) licenser startar en inspelning av Teams-möte, ändras detta.  

Mer information finns i [Ladda upp en inspelning med Microsoft Teams-möte till direkt uppspelning](https://docs.microsoft.com/stream/portal-upload-teams-meeting-recording).
