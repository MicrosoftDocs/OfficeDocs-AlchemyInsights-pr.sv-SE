# <a name="not-receiving-facebook-codes-using-outlookcom"></a>Får inte Facebook-koder med Outlook.com

Om du begärde ett nytt lösenord från Facebook men inte fick ditt e-postmeddelande om återställning av lösenord:

- Kontrollera din [skräppostmapp](https://outlook.live.com/mail/junkemail).
- Försök att [återställa ditt Facebook-lösenord igen](https://aka.ms/facebook-password-reset).

Om du inte kan återställa lösenordet på grund av att du inte kan komma åt e-postmeddelandet eller mobiltelefonnumret på ditt konto provar du [Facebooks tips om återställning av lösenord](https://aka.ms/facebook-password-help).
