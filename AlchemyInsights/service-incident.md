# <a name="emerging-issue"></a>Problem med framväxande

Flera kunder har klagat till ett problem som liknar det du rapporterar. Vi tittar i det här problemet.

Kontrol lera [instrument panelen för tjänstens hälsa](https://admin.microsoft.com/adminportal/home#/servicehealth) för att uppdatera din tjänst som påverkar händelser.
