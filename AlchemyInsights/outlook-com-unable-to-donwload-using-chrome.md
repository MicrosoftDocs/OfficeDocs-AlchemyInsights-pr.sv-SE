# <a name="cant-download-files-in-chrome"></a>Det går inte att ladda ned filer i Chrome

**Problem** Vi är medvetna om ett problem som förhindrar att vissa kunder kan ladda ned filer när du använder Google Chrome-webbläsaren. 

**Lösning** Använd [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) eller en annan webbläsare.
Läs mer om hur du [laddar ned filer och Outlook.com bifogade](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy)filer.
