# <a name="how-do-i-clear-the-cache-or-locally-resetrefresh-the-clientapp"></a>Hur rensar jag cacheminnet eller lokalt återställa/uppdatera klienten/appen?

Om du ser ofullständiga eller felaktiga data, försök att tvinga stänga och öppna appen.  

Om det inte fungerar kan du försöka återställa kontot på enheten.
 
Öppna appen, gå till **Inställningar**, Välj e-postkontot och välj **Återställ konto**.
