# <a name="how-do-i-clear-the-cache-or-locally-resetrefresh-the-clientapp"></a>Hur rensar jag cacheminnet eller återställer/uppdaterar du klient/app lokalt?

Om du ser ofullständiga eller felaktiga data kan du försöka tvinga fram stängning och öppna appen igen.  

Om det inte fungerar kan du försöka återställa kontot på enheten.
 
Öppna appen, gå till **Inställningar**, Välj e-postkontot och välj **Återställ konto**.
