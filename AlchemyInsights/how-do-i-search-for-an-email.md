# <a name="how-do-i-search-for-an-email"></a>Hur söker jag efter ett e-postmeddelande?

Du kan söka efter ett e-postmeddelande genom att leta upp kontakten som skickade e-postmeddelandet eller genom att söka efter ett nyckelord som finns i e-postmeddelandet.

Tryck på förstoringsglaset i det nedre navigeringsfältet. Skriv sedan din sökbegäran i rutan längst upp på skärmen. 

**Tips:** Du kan dubbelknacka på förstoringsglaset för att få fokus på sökrutan. Dina resultat visas när du skriver din begäran.
