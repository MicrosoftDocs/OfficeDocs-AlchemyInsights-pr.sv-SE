# <a name="change-my-profile-picture-or-name-in-outlookcom"></a>Ändra min profil bild eller ett namn i Outlook.com

1. Gå till [ditt Microsoft-konto](https://go.microsoft.com/fwlink/p/?linkid=860841).
1. Välj **Redigera namn** eller **ändra bild**.

Du kan göra upp till 24 timmar för att ändra din profil bild eller ditt namn.
