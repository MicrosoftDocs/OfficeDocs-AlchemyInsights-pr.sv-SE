# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Version av SharePoint-migreringsverktyget som stöds



SharePoint-migreringsverktyget stöder för närvarande endast migreringar från SharePoint 2013. Versionerna 2010 och 2016 stöds inte vid denna tidpunkt.
  
Mer information finns [i Introduktion till SharePoint-migreringsverktyget](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
