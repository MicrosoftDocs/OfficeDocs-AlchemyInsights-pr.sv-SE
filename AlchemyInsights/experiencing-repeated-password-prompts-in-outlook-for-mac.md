# <a name="experiencing-repeated-password-prompts-in-outlook-for-mac"></a>Har du upprepade frågor om lösen ord i Outlook för Mac?

Vi kan erbjuda mer specialiserad hjälp för Outlook för Mac direkt från programmet.
  
Gå till den berörda användarens Mac, öppna Outlook och klicka på **Hjälp** \> **kontakta support**.
  
Mer information om hur du använder kund support i Outlook för Mac-appen finns i den här artikeln: [ny kontakta support-funktion i Outlook 2016 för Mac](https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac-mso_mac2016/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517.aspx)
