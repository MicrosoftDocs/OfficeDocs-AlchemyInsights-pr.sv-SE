# <a name="experiencing-repeated-password-prompts-in-outlook-for-mac"></a>Får du upprepade uppmaningar om lösenord i Outlook för Mac?

Vi kan hjälpa dig med mer specialiserad hjälp för Outlook för Mac direkt från programmet.
  
Gå till den aktuella användarens Mac, öppna Outlook och klicka på **Hjälp för** att \> **kontakta support.**
  
Mer information om hur du använder kundsupport i appen Outlook för Mac finns i den här artikeln: Ny kontakta support-funktion i [Outlook 2016 för Mac](https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac-mso_mac2016/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517)
