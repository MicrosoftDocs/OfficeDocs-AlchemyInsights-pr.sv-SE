# <a name="experiencing-repeated-password-prompts-in-outlook-for-mac"></a>Drabbats av upprepade uppmaningar lösenord i Outlook för Mac?

Vi kan bidra till mer specialiserad Outlook för Mac Hjälp direkt från programmet.
  
Gå till den aktuella användarens Mac, öppna Outlook och klicka på **Hjälp om** \> **Kontakta Support**.
  
Mer information om hur du använder Outlook för Mac-app Kundsupport finns i den här artikeln: [Ny kontakt stöder funktionen i Outlook 2016 för Mac](https://support.office.com/article/https://answers.microsoft.com/en-us/msoffice/forum/msoffice_outlook-mso_mac-mso_mac2016/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517.aspx)
