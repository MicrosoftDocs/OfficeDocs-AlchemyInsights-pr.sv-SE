# <a name="access-services"></a>Access Services

Funktionen Access Services i SharePoint Online avvecklas. Nya Access-baserade webb program och webb databaser kan inte längre skapas och befintliga webb program och webb databaser stängs av i april 2018. Access-databaser (. accdb) påverkas inte av den här ändringen. Vi rekommenderar Microsoft PowerApps som ett alternativ till att komma åt webbappar och webb databaser. 
  
[Lär dig mer om Microsoft PowerApps](https://powerapps.microsoft.com/)
