# <a name="access-services"></a>Access Services

Access Services-funktion i SharePoint Online dras. Nya Access-baserade web apps och webbdatabaser kan inte längre skapas och befintliga web apps och webbdatabaser kommer att stängas av April, 2018. Desktop Access-databaser (.accdb) påverkas inte av ändringen. Vi rekommenderar Microsoft PowerApps som ett alternativ till Access web apps och webbdatabaser. 
  
[Lär dig mer om Microsoft PowerApps](https://powerapps.microsoft.com/)
