# <a name="subscription-notice-in-your-office-client"></a>Prenumerations meddelande i Office-klienten?

Det verkar som om du ser ett meddelande i Office-klienten som är relaterad till prenumerationens status. Information om hur du åtgärdar problemet i den här artikeln: [ett prenumerations meddelande visas när jag öppnar ett Microsoft 365-program](https://support.office.com/article/A-subscription-notice-appears-when-I-open-an-Office-365-application-4cabe32c-f594-4c0e-9191-3d3ade10cceb.aspx)
  
Om ditt abonnemang är uppdaterat kan du ha ringt ett tillfälligt problem som gjorde att din Office-aktivering inte fungerar. Det finns några enkla åtgärder du kan vidta för att undvika dessa meddelanden, till exempel kontrol lera datorns tid, köra Office som administratör och logga in på activate. Detaljerade anvisningar för hur du åtgärdar problemen i den här artikeln: [olicensierad produkt och aktiverings fel](https://support.office.com/article/Unlicensed-Product-and-activation-errors-in-Office-0d23d3c0-c19c-4b2f-9845-5344fedc4380.aspx).
  
Om de inte fungerar för dig, var god kontakta oss med länkarna nedan så är vi glada att hjälpa dig.
