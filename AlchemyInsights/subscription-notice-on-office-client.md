# <a name="subscription-notice-in-your-office-client"></a>Prenumerationsmeddelande i Office-klienten?

Det ser ut som om du ser ett meddelande i Office-klienten som är relaterad till prenumerationens status. Du hittar information om hur du åtgärdar problemet i den här artikeln: [Ett prenumerationsmeddelande visas när jag öppnar ett Microsoft 365-program](https://support.office.com/article/A-subscription-notice-appears-when-I-open-an-Office-365-application-4cabe32c-f594-4c0e-9191-3d3ade10cceb.aspx)
  
Om din prenumeration är uppdaterad kan du ha drabbats av ett tillfälligt problem som gjorde att Office-aktiveringen misslyckades. Det finns enkla åtgärder du kan vidta för att undvika dessa meddelanden, till exempel kontrollera datorns tid, köra Office som administratör och logga in för att aktivera. Du kan hitta detaljerade steg för att åtgärda dessa problem i den här artikeln: [Olicensierade produkt- och aktiveringsfel](https://support.office.com/article/Unlicensed-Product-and-activation-errors-in-Office-0d23d3c0-c19c-4b2f-9845-5344fedc4380.aspx).
  
Om dessa inte fungerar för dig, vänligen kontakta oss med länkarna nedan och vi kommer gärna att hjälpa till.
