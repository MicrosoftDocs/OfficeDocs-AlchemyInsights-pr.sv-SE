# <a name="how-to-get-support-in-outlookcom"></a>Hur du får support på Outlook.com

Tyvärr, Outlook.com erbjuder inte telefonsupport, men vi har ett dedikerat team som kan hjälpa dig.
Ange din fråga beskriver ditt problem i rutan **Hjälp** ovan och välj **Hjälp**.
