# <a name="duplicate-object-synchronization-errors"></a>Dubblettobjekt synkroniseringsfel

Om du får något av följande fel under synkroniseringen:

- Det finns redan ett synkroniserat objekt med samma proxyadress i Microsoft Online Services-katalogen.

- Ett liknande meddelande för UserPrincipalName.

Använda den nya diagnostiska AADConnect hälsa för att lösa dubblettobjekt fel. Mer information finns i [diagnostisera och åtgärda synkroniseringsfel duplicerade attributet](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).
