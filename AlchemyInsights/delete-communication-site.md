# <a name="delete-a-communication-site"></a>Ta bort en webbplats för kommunikation

1. Gå till webbplatsen.
    
2. Klicka på ikonen inställningar i det övre högra hörnet.
    
3. Klicka på **Webbplatsinställningar**. Klicka på **Ta bort den här webbplatsen**under **Webbplatsinformation**och klicka sedan på **Ta bort**.
