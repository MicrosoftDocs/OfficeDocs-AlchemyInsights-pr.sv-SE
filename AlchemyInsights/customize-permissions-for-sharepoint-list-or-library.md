# <a name="how-to-customize-permissions-for-a-sharepoint-list-or-library"></a>Anpassa behörigheter för en SharePoint-lista eller ett bibliotek

Som administratör eller ägare av ett bibliotek, en lista eller en undersökning kan du ändra behörigheterna för att låta en grupp personer komma åt de data de behöver medan de begränsar andra användare.
