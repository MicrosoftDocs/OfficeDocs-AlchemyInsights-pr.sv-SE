# <a name="why-cant-i-edit-this-file"></a>Varför kan jag inte redigera den här filen?

Om ett dokument, kalkyl blad eller en presentation är låst för redigering och inte kan redige ras i Windows-skrivbordet kan det bero på att filen redige ras av någon annan eller att filen har checkats ut.

Ta en titt på några av orsakerna till [varför du kanske inte kan redigera filen](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).
