# <a name="change-the-domain-for-microsoft-365-group"></a>Ändra domänen för Microsoft 365 Group

Om du vill ändra domänen för en Microsoft 365-grupp när du skapar följer du stegen i [Välj den domän som ska användas när du skapar Microsoft 365 Groups](https://docs.microsoft.com/office365/admin/create-groups/choose-domain-to-create-groups).

När Microsoft 365-gruppen redan har skapats finns det för närvarande inget sätt att ändra domänen. Det enda sättet är att skapa en ny Microsoft 365-grupp och ta bort den gamla om det behövs (ytterligare e-postadresser kan läggas till via [Exchange Admin Center](https://outlook.office365.com/ecp).
