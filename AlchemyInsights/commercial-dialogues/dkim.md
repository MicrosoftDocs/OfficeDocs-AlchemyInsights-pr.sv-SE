# <a name="dkim-domainkeys-identified-mail"></a>DKIM (DomainKeys Identified Mail)

DKIM (DomainKeys Identified Mail) är ett protokoll som gör att en organisation kan ta ansvar för att överföra ett meddelande på ett sätt som kan verifieras av postlådeleverantörer. Den här verifieringen görs möjlig genom kryptografiska autentisering.
