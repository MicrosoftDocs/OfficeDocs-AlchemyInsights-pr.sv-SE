# <a name="control-the-access-from-unmanaged-devices"></a>Styra åtkomsten från ohanterade enheter

Som SharePoint-administratör eller global administratör i Microsoft 365 kan du blockera eller begränsa åtkomsten till SharePoint och OneDrive-innehåll från ohanterade enheter. Mer information finns i [Styra åtkomsten från ohanterade enheter](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices).
