# <a name="changing-shared-mailbox-send-as-behavior"></a>Ändra delad post låda skicka som-beteende

Standardinställningen är att e-post skickad från den delade postlådan inte sparas i den gemensamma Skickat-mappen. Den sparas istället hos den person som skickade meddelandet, i mappen Skickat.
  
Om du vill att alla medlemmar ska kunna se det skickade e-postmeddelandet redigerar du informationen om den delade post lådan i administrations centret för Microsoft 365 och väljer **Sent items** \> **Redigera**skickat.
  
Mer information finns i [skapa en delad post låda](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox).
