# <a name="change-your-microsoft-account-picture"></a>Ändra din Microsoft-profil

1. Gå till [ditt Microsoft-konto](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. Välj **ändra bild**.
3. Välj **ny bild**.
4. Välj den bild du vill använda och välj **Öppna**.
5. Dra eller ändra storlek på bild rutan och välj **Spara**.

Låt upp till 24 timmar innan ändringar i ditt profil bild träder i kraft.
