# <a name="profile-property-synchronization"></a>Synkronisering av profilegenskap

Det går inte att mappa ytterligare egenskaper för AD Import för att synkronisera från Azure Active Directory till användarprofilprogrammet. AD Import är begränsad till en förkonfigurerad uppsättning egenskaper för att garantera konsekvent prestanda för timerjobbet.
  
[Läs mer om profilegenskaperna som synkroniseras](https://go.microsoft.com/fwlink/?linkid=875671)
