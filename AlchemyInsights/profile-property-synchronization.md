# <a name="profile-property-synchronization"></a>Synkronisering av profil egenskap

Det går inte att mappa ytterligare egenskaper för AD-import till synkronisering från Azure Active Directory till användar profil programmet. AD-import är begränsad till en förkonfigurerad uppsättning egenskaper för att garantera konsekvent prestanda för det tidsinställda jobbet.
  
[Läs mer om profil egenskaper som synkroniseras](https://go.microsoft.com/fwlink/?linkid=875671)
