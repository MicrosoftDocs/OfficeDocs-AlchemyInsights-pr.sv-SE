# <a name="license-missing-or-disappears-from-the-admin-center"></a>Licens saknas eller försvinner från administrationscentret


Om dina licenser försvann kontrollerar du att prenumerationen inte har upphört att gälla. Om du vill kontrollera utgångsdatumet går du till **Fakturering**   >   **[dina produkter](https://go.microsoft.com/fwlink/p/?linkid=842054)**.  

Beroende på vyn visas utgångsdatumet längst till höger med etiketten **Datum** eller i **avsnittet Fakturering** i prenumerationsinformationen **bredvid Förfallodagen på**.
