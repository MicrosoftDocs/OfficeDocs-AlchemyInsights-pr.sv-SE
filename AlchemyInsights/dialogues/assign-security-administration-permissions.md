# <a name="assign-the-security-administration-permissions"></a>Tilldela behörigheter för säkerhetsadministration

1. Välj Behörigheter > Säkerhetsadministratör i Säkerhets- & **Office 365.**
2. Klicka på **Redigera bredvid** Medlemmar i **fönstret** **Säkerhetsadministratör.**
3. Klicka på **Välj medlemmar i** fönstret Redigera val av **medlemmar.**
4. Klicka **på + Lägg** till i **fönstret** Välj medlemmar och välj sedan användarnas namn.
5. Välj **Lägg till > Klar > Spara.**
