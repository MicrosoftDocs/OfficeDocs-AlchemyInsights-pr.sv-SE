# <a name="permissions-and-browser-related-requirements-for-microsoft-365-defender"></a>Behörigheter och webbläsarrelaterade krav för Microsoft 365 Defender

- Behörigheter: Om du vill aktivera Microsoft 365 Defender måste du vara global administratör eller säkerhetsadministratör i Azure Active Directory. I Hantera åtkomst till Microsoft [365 Defender](https://go.microsoft.com/fwlink/?linkid=2143626)finns en lista över roller som kan använda Microsoft 365 Defender och hur åtkomst till data måste regleras.

- Webbläsarrelaterade krav: Om du vill få åtkomst till Microsoft 365 Defender i Säkerhetscenter för Microsoft 365 använder du Microsoft Edge, Internet Explorer 11 eller en HTML 5-kompatibel webbläsare.
