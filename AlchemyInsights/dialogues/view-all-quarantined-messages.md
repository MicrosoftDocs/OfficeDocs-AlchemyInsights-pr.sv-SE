# <a name="view-all-quarantined-messages"></a>Visa alla meddelanden i karantän

1. I administrationscentret går du till [Säkerhets- & Efterlevnadscenter.](https://go.microsoft.com/fwlink/p/?linkid=2077143)
2. Gå till **karantän för hantering** av  >    >  **hot.**
3. Som standard sorteras meddelanden från nyaste till äldsta baserat på datumet då meddelandet togs emot. Du kan sortera på ett fält genom att klicka på ett sidhuvud.
4. Du kan bara massåtgärder på upp till 100 objekt, så filtrering kan också minska resultatuppsättningen om du har fler än 100 objekt.
