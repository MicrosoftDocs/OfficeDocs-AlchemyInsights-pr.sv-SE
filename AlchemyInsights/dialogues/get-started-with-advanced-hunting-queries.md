# <a name="get-started-with-advanced-hunting-queries"></a>Komma igång med avancerade sökfrågor

Microsoft-säkerhetsvakterna delar [regelbundet avancerade sökningsfrågor i GitHub.](https://go.microsoft.com/fwlink/?linkid=2144624)

De här frågorna erbjuder också avancerade sökfrågor för att hitta aktiviteter och indikatorer associerade med [nya hot.](https://go.microsoft.com/fwlink/?linkid=2145808) Dessa frågor tillhandahålls som en del av rapporterna om hotanalyser i Microsoft Defender Säkerhetscenter.
