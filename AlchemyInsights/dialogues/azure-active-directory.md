# <a name="azure-active-directory"></a>Azure Active Directory

Microsoft Azure Active Directory (Azure AD) är en molnbaserad tjänst för identitets- och åtkomsthantering som hjälper anställda att logga in och få åtkomst till resurser både externt och internt. Mer information finns i följande resurser:

- [Vad är Azure Active Directory?](https://go.microsoft.com/fwlink/?linkid=2081145)
- [Azure-integrering med Office 365](https://go.microsoft.com/fwlink/?linkid=2081218)
