# <a name="open-a-second-browser-window"></a>Öppna ett andra webbläsarfönster

Så här öppnar du ett andra webbläsarfönster:

1. I det första webbläsarfönstret (där du läser det här) ser du till att domänguiden är öppen. Om du inte klickade på knappen ovan för att starta domänguiden kan du gå till domänguiden genom att gå till **Domäner > Lägg till domän.**

2. Öppna ett till webbläsarfönster. I den andra webbläsaren följer du nästa steg.
