# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>Introducera en Windows 10-enhet till Microsoft Defender Advanced Threat Protection (MDATP)

MDATP har stöd för följande distributionsverktyg och metoder för Windows 10:

- grupprincip
- Microsoft Endpoint Configuration Manager
- Hantering av mobila enheter, inklusive Microsoft Intune
- Lokala skript

Mer information finns i [Onboarding-verktyg och -metoder för Windows 10-enheter.](https://go.microsoft.com/fwlink/?linkid=2143460)
