# <a name="tips-about-recalling-messages"></a>Tips om att återkalla meddelanden

Det här gör du när du kan återkalla meddelanden:

* Som administratör kan du återkalla meddelanden åt användare med hjälp av PowerShell. Du kan inte återkalla meddelanden från administrationscentret.
* Du kan bara återkalla meddelanden som skickas till personer i organisationen. Om meddelandet till exempel skickades till en Gmail-adress kan du inte återkalla det.
* **Du kan bara återkalla meddelanden som skickats från Outlook 2016 för PC.** Om en användare skickar ett meddelande med Outlook för Mac eller Outlook på webben kan du inte återkalla det.
