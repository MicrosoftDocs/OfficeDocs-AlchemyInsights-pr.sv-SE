# <a name="setting-up-visio"></a>Konfigurera Visio

Det verkar som om du behöver hjälp med att konfigurera Visio med ditt Microsoft 365-abonnemang.
  
Meddelande för Mac-användare: Visio-produkten är inte tillgänglig för MacOS för närvarande. Om du har en dator bör du kunna installera program varan enligt anvisningarna nedan.
  
Du kan följa stegen i [denna artikel](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710.aspx) för att slutföra konfigurationen.
