# <a name="setting-up-visio"></a>Ställa in Visio

Du behöver hjälp med att konfigurera Visio med din Microsoft 365-prenumeration.
  
Meddelande för Mac-användare: Visio-produkten är inte tillgänglig för MacOS just nu. Om du har en dator bör du kunna installera programvaran med hjälp av stegen nedan.
  
Du kan följa stegen i [denna artikel](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710.aspx) för att slutföra konfigurationen.
