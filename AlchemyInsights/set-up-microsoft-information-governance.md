# <a name="set-up-microsoft-information-governance"></a>Konfigurera Microsoft informations styrning

[Installations guiden för Microsoft information styrelse](https://admin.microsoft.com/AdminPortal/Home#/modernonboarding/migsetupguide) tillhandahåller den information du behöver för att konfigurera och hantera organisationens styrnings strategi, för att säkerställa att dina data klassificeras och hanteras enligt de särskilda rikt linjer för livs cykeln som du anger.

Med den här guiden får du lära dig hur du skapar, lägger till eller publicerar etiketter, etikett principer och bevarande principer som tillämpas på organisationens åter användnings Bart innehåll och efterlevnad. Du får också information om hur du importerar CSV-filer med en fil plan för flera olika scenarier eller för att tillämpa dem manuellt på enskilda dokument.
