# <a name="change-the-country-or-region-for-your-microsoft-365-subscription"></a>Ändra land eller region för Microsoft 365-prenumerationen

Du **kan inte ändra** land eller region för abonnemanget. Det beror på att det land eller den region där din organisation är sitt huvud kontor bestämmer vilka tjänster som är tillgängliga för dig, moms och fakturerings valuta samt data centrets plats. Om du vill ändra organisationens land eller region kan du registrera dig för ett nytt konto, välja önskat land eller region och köpa ett nytt abonnemang.

Mer information finns i [ändra organisationens adress, tekniska kontakt med mera](https://docs.microsoft.com/microsoft-365/admin/manage/change-address-contact-and-more?view=o365-worldwide). Om du vill ändra den associerade fakturerings-eller prenumerations adressen kan du läsa [ändra fakturerings adresser för Microsoft 365 för företag](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/change-your-billing-addresses?view=o365-worldwide).
