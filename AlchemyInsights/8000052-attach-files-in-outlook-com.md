# <a name="how-to-attach-files"></a>Hur du bifogar filer

Bifoga filer i Outlook.com klassisk:
  
1. Skapa ett nytt meddelande eller svarar på ett meddelande.
    
2. Välj **Bifoga**högst upp i rutan Skriv. 
    
Bifoga filer i Outlook.com-beta:
  
1. Skapa ett nytt meddelande eller svarar på ett meddelande.
    
2. Längst ned i fönstret Skriv Välj ![Bifoga](media/da223d01-5fe6-448c-a3a3-e2b5262da4b9.png).
    
Mer information om [hur du bifogar filer i Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=2001702&amp;clcid=0x409).
