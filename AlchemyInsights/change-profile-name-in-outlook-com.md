# <a name="change-your-profile-name"></a>Ändra ditt profilnamn

1. Gå till [Din Microsoft-kontoinformation](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. Välj **Redigera namn**. 
    
3. Ange ett för- och efternamn och välj **Spara**. 
    
Det kan ta upp till 24 timmar innan ändringar i ditt profilnamn börjar gälla.
  
Mer information finns [i Ändra profilfoto eller namn i Outlook.com](https://go.microsoft.com/fwlink/?linkid=873110).
