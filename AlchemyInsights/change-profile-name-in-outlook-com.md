# <a name="change-your-profile-name"></a>Ändra ditt profil namn

1. Gå till [ditt Microsoft-konto](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. Välj **Redigera namn**. 
    
3. Ange förnamn och efter namn och välj **Spara**. 
    
Låt upp till 24 timmar få ändringar i ditt profil namn att träda i kraft.
  
Mer information finns i [ändra din profil bild eller ditt namn i Outlook.com](https://go.microsoft.com/fwlink/?linkid=873110).
