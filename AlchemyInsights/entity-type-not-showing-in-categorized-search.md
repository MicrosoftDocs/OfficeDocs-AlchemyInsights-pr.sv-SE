# <a name="entity-type-not-showing-in-search-results"></a>Typ av enhet inte visas i sökresultat

För en entitetstyp i kategoriserade Sök eller betydelse för sökning i Dynamics 365 Unified gränssnitt, måste det först läggas till modulen app.

Anmärkning: När du söker efter bifogade filer i betydelse för sökning, entiteten bifogad fil måste läggas till modulen app även om den överordnade posten är vad som visas i resultaten.
