# <a name="inbox-not-loading-with-firefox"></a>Inkorgen inte läses in med Firefox

Vissa tillägg i webbläsaren Firefox hindrar Inkorgen Outlook.com från att läsas in.
  
Besök Firefox-stöd för att lära dig hur du [inaktiverar eller ta bort tillägg](https://support.mozilla.org/kb/disable-or-remove-add-ons).
