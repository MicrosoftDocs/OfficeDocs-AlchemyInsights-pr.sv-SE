# <a name="permissions-and-public-folders"></a>Behörigheter och gemensamma mappar

Du kan ändra behörigheterna för dina gemensamma mappar med Outlook, administrations centret för Exchange (UK) eller PowerShell:
  
- [Klicka här](https://support.office.com/article/Set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005.aspx)för Outlook-instruktioner.
    
- Anvisningar för UK finns i [den här artikeln](https://technet.microsoft.com/library/jj651147%28v=exchg.150%29.aspx.aspx#Anchor_1) . 
    
- Information om hur du använder PublicFolderClientPermission cmdleten finns i [den här artikeln](https://technet.microsoft.com/library/bb124743%28v=exchg.160%29.aspx.aspx) för PowerShell. Om du behöver instruktioner för att ansluta till Exchange PowerShell klickar du [här](https://technet.microsoft.com/library/jj984289%28v=exchg.160%29.aspx.aspx).
    
Om **externa användare inte kan skicka e-post till en gemensam e-postmapp**kan orsaken vara att den gemensamma mappen saknar behörigheter som krävs för extern e-postleverans. Du kan åtgärda detta med hjälp av Outlook-instruktionerna [här](https://technet.microsoft.com/library/aa997560%28v=exchg.150%29.aspx.aspx#Anchor_1)eller PowerShell-instruktionerna [här](https://support.microsoft.com/help/2984402/-5.7.1-smtp-550-5.7.1-resolver.rst.authrequired-nondelivery-report-when-external-users-try-to-send-mail-to-mail-enabled-public-folders-in-office-365.aspx).
