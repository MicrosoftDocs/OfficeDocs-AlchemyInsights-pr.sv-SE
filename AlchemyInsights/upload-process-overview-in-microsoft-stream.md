# <a name="upload-process-overview-in-microsoft-stream"></a>Översikt över överföringsprocesser i Microsoft Stream

När du laddar upp en video kodar Microsoft Stream automatiskt in den i flera bithastigheter så att videon kan strömmas med hjälp av adaptiv bithastighetsströmning. Microsoft Stream optimerar bithastigheten och kvaliteten på den uppspelningsvideo som spelas upp baserat på tittarens tillgängliga bandbredd.

- [Översikt över överföringsprocesser i Microsoft Stream](https://docs.microsoft.com/stream/upload-process-overview)
