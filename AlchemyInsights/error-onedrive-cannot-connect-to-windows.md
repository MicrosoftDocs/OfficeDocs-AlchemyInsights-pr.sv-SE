# <a name="error-onedrive-cannot-connect-to-windows"></a>Fel: OneDrive kan inte ansluta till Windows

Sök efter filen cldflt.sys att se om den körs och aktivera om den inte löser felmeddelandet. 
  
Mer information om hur du aktiverar filen cldflt.sys finns i [OneDrive kan inte ansluta till Windows ”fel vid åtkomst till filer i OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
