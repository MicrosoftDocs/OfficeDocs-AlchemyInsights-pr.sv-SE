# <a name="cant-add-or-change-teams-profile-picture"></a>Det går inte att lägga till eller ändra profil bild för team

Om du vill lägga till eller ändra en profil bild måste användarna vara fullständigt aktiverade med följande:

- Exchange Online
- SharePoint Online
- Möjlighet att skapa Microsoft 365-grupper

Se team administratören om att få nödvändiga behörigheter. Anvisningar finns i [ändra din bild i Teams](https://support.microsoft.com/office/change-your-picture-in-teams-7a711943-9248-420e-b814-c071aa8d9b9c).
