# <a name="questions-about-your-microsoft-365-license"></a>Frågor om din Microsoft 365-licens

Om du vill veta hur många licenser du har köpt eller hur många licenser som tilldelas från en prenumeration:
  
1. Gå till sidan \> **[Faktureringsprodukter](https://go.microsoft.com/fwlink/p/?linkid=842054)** i administrationscentret. **Billing**

2. Hitta den prenumeration du vill veta mer om. Du kan se antalet licenser du betalar för och hur många som är tillgängliga för att tilldelas.

Om du vill lägga till fler licenser, eller om du har för många, väljer du **Lägg till/ta bort licenser** för att justera det köpta beloppet.
