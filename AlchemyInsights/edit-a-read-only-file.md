# <a name="edit-a-read-only-file"></a>Redigera en skrivskyddad fil

Om du klickar på ett dokument namn i dokument bibliotek kan Office-programmet oväntat öppnas som en skrivskyddad fil. Öppna den för redigering genom att klicka på **Öppna** (tre punkter) och sedan på **Redigera.**
