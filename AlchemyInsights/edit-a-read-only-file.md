# <a name="edit-a-read-only-file"></a>Redigera en skrivskyddad fil

När du klickar på ett dokuments namn i dokumentbibliotek kan Office-programmet oväntat öppna det som en skrivskyddad fil. Om du vill öppna den för redigering klickar du på **Öppna** (de tre punkterna) och sedan på **Redigera.**
