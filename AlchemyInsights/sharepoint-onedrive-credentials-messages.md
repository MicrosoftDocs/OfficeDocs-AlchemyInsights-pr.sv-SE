# <a name="credential-messages-in-sharepoint-or-onedrive"></a>Inloggningsinfo-meddelanden i SharePoint eller OneDrive

Användarna kan med jämna mellanrum uppmanas att ange Office för autentiseringsuppgifter till SharePoint Online eller OneDrive när de försöker öppna dokument.

Om du upplever dessa problem kan du prova stegen i [den här artikeln](https://support.microsoft.com/help/2913639/office-applications-periodically-prompt-for-credentials-to-sharepoint).

Om du använder enkel inloggning (SSO) för att autentisera kan du i stället läsa [den här artikeln](https://support.microsoft.com/help/4025962/cant-sign-in-after-update-to-office-2016-build-16-0-7967-on-windows-10) .
