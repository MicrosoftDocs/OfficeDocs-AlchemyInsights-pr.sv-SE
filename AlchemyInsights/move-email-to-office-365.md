# <a name="move-email-to-microsoft-365"></a>Flytta e-post till Microsoft 365

Du kan migrera e-post till Microsoft 365 på flera olika sätt. Med [Microsoft 365-guiden för e-postmigrering](https://aka.ms/alchemyinsight-mailmigrationadvisor) kan du välja det bästa för din miljö och vägleda dig genom e-postmigreringen.
  
Om du vill läsa om dina alternativ utan att starta migreringen kan du läsa ange [en sökväg för migrering](https://docs.microsoft.com/Exchange/mailbox-migration/decide-on-a-migration-path).

Du kan också gå till sidan [anta Microsoft 365 med FastTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) om du behöver hjälp med migreringen.
