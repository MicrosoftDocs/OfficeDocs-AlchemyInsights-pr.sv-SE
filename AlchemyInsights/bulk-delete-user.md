# <a name="bulk-delete-user"></a>Användare av massborttagning

Du kan ta bort ett stort antal användare från gruppen med hjälp av en CSV-fil (Kommaavgränsade värden) för att ta bort användare i massutklädnad. Information om hur du tar bort massanvändare med Azure AD-portal finns [i Massborttagning av användare i Azure Active Directory](https://docs.microsoft.com/azure/active-directory/users-groups-roles/users-bulk-delete).
