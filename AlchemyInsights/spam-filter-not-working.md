# <a name="spam-filter-not-working"></a>Skräp post filtret fungerar inte

Om du tror att ett meddelande som du har fått i Outlook.com är ett phishing-eller Spoof **bedrägeri markerar du meddelandet och markerar** \> **Phishing** det som ett phishing-bedrägeri. Om du vill veta mer om missbruk, nätfiske och förfalskningar [läser du detta](https://support.office.com/article/0d882ea5-eedc-4bed-aebc-079ffa1105a3?wt.mc_id=Office_Outlook_com_Alchemy). Om du vill blockera e-postadresser, lägga till adresser eller domäner i listan Betrodda avsändare eller använda andra alternativ för skräp post väljer du **skräp post alternativ**.
