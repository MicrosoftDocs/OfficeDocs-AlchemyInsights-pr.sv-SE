# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Skaffa ett konto inaktiveras fel i Outlook på webben?

Om du får felet **ditt konto har inaktiverats** med `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` när du försöker öppna Outlook på webben (kallades tidigare Outlook Web App) organisationens-administratör kan ha inaktiverat din åtkomst till Outlook på webben. Administratören kan åtgärda felet genom att följa anvisningarna i avsnittet [Aktivera eller inaktivera Outlook Web App för en postlåda](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
