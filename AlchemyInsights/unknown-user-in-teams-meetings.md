# <a name="unknown-user-appears-in-teams-meetings-or-chats"></a>"Okänd användare" visas i Teams möten eller chattar

Användaren kan ha tagits bort från Microsoft 365. I det här fallet kan du försöka ta bort användaren från Teams.  

Prova att rensa följande cache när Teams är stängda och starta sedan om Teams: `C:\Users\<USER>\AppData\Roaming\Microsoft\Teams`

Uppdateringar av det här problemet finns i  [borttagen användare visas som "okänd användare" i Microsoft Teams](https://docs.microsoft.com/MicrosoftTeams/troubleshoot/known-issues/removed-user-appears-as-unknown).
