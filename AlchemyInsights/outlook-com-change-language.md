# <a name="change-your-language-date-and-time-format-or-time-zone-in-outlookcom"></a>Ändra språk, datum-och tids format eller tidszon i Outlook.com

1. Öppna [Inställningar för språk och tid](https://go.microsoft.com/fwlink/?linkid=2085505).
1. Välj språk, datum format, tids format och tidszon som du vill använda.
1. Välj **Spara**.
