# <a name="subscription-needed-for-microsoft-edge-sync-to-work-with-azure-active-directory-accounts"></a>Abonnemang krävs för Microsoft Edge Sync för att fungera med Azure Active Directory-konton

Synkroniseringen fungerar med någon av följande prenumerationer:

1. Azure Active Directory Premium (P1 och P2)
1. Microsoft 365 Business Premium
1. Office 365 E3 och senare
1. Azure information Protection (P1 och P2)
1. Alla Office 365 Education-prenumerationer:
    - Microsoft-program för elever eller lärare
    - Exchange Online för elever eller lärare
    - Office 365 a1 eller senare
    - Microsoft 365 a1 eller senare
    - Azure information Protection P1 eller P2, för elever eller lärare
