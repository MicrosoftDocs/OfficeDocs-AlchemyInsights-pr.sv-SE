# <a name="install-and-configure-onedrive"></a>Installera och konfigurera OneDrive

OneDrive sync-klient är förinstallerad med Windows 10. Installera OneDrive för Windows 7 eller Mac eller få mobil app för din telefon eller tablet, gå till [http://onedrive.com/download](http://onedrive.com/download).
  
Om du vill konfigurera OneDrive sync-klient i Windows finns:
  
- [Ställ in datorn för att synkronisera din OneDrive för Business-filer i Office 365](https://go.microsoft.com/fwlink/?linkid=533375)

- [Synkronisera SharePoint-filer med den nya OneDrive-synkroniseringsklienten](https://go.microsoft.com/fwlink/?linkid=871666)
