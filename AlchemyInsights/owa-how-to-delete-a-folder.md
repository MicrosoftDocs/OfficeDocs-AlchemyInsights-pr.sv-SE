# <a name="how-to-delete-a-folder-in-outlook-on-the-web"></a>Ta bort en mapp i Outlook på webben

1. Högerklicka på den mapp eller undermapp som du vill ta bort i mappfönstret.
2. Välj **Ta bort mapp**.
3. Välj **OK** för att bekräfta.

Mer information finns [i Arbeta med meddelandemappar i Outlook på webben](https://support.office.com/article/ae0f10d6-54e7-4f29-acd3-78cdc3fdcb9f).
