# <a name="learn-about-external-calendar-sharing-options"></a>Läs mer om alternativ för extern kalenderdelning

Om du behöver schemalägga möten med personer utanför organisationen kan du med Microsoft 365 göra kalendrar tillgängliga för "externa användare", de som behöver se ledig/upptagen tid men inte har användarkonton för din Microsoft 365-miljö. Mer information finns i [Dela kalendrar med externa användare](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).
