# <a name="learn-about-external-calendar-sharing-options"></a>Läs mer om externa kalenderdelningsalternativ

Om du behöver schemalägga möten med personer utanför organisationen kan du med Office 365 göra kalendrar tillgängliga för externa användare, de som behöver se ledig/upptagen tid men inte har användarkonton för Office 365-miljön. Mer information finns i [dela kalendrar med externa användare](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).
