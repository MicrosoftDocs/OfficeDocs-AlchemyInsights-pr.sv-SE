# <a name="modern-commerce-administrator-role"></a>Rollen för modern handelsadministratör

Den här rollen tilldelas automatiskt till användare som gör ett köp för att ge dem begränsad åtkomst till administratörsportalen för att hantera det köpet. Dessa användare är inte administratörer och har inte förhöjda privilegier.
