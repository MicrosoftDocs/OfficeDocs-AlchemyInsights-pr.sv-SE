# <a name="import-contacts"></a>Importera kontakter

Innan du importerar kontakter till Outlook.com måste du exportera dem från din andra e-postleverantör som en CSV-fil. Följ instruktionerna nedan.
  
1. Och välj **personer**längst ned på sidan. 
    
2. På menyraden, Välj **Hantera** \> **Importera kontakter**. 
    
3. Välj **Gmail** (även om CSV-filen inte kan komma från Gmail) eftersom Gmail använder CSV-filtyp. 
    
4. Välj **Bläddra**, navigera till mappen där du sparade exporterade kontakterna och välj **Öppna**. 
    
Mer information finns i [Importera kontakter till ett Outlook.com-konto](https://go.microsoft.com/fwlink/p/?linkid=873136).
