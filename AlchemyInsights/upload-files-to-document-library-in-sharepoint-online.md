# <a name="upload-files-to-onedrive-or-sharepoint"></a>Överföra filer till OneDrive eller SharePoint

Du kan överföra filer i ett dokumentbibliotek i SharePoint Online genom att dra dem från datorn och släppa dem i dokumentbiblioteket. Med OneDrive, kan du synkronisera filer mellan din dator och molnet, så att du kan öppna dina filer från valfri plats. Du kan också arbeta med dina synkroniserade filer direkt i File Explorer och komma åt dina filer även när du är offline. Mer information om att överföra och synkronisera finns i länkarna nedan.

- [Överför filer eller en mapp i ett bibliotek](https://support.office.com/article/upload-a-folder-or-files-to-a-document-library-eb18fcba-c953-4d45-8d90-8da66edeacdb)

- [Överföra med synkronisering](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-in-windows-615391c4-2bd3-4aae-a42a-858262e42a49)
