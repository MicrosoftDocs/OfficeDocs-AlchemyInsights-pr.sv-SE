Har du problem med att starta Outlook 2016, Outlook 2013 eller Outlook 2010 eller ta emot "kan inte starta Microsoft Outlook. Kan du inte öppna Outlook-fönstret "fel meddelanden?
  
Börja med att lösa problemen genom att använda fel söknings stegen i [den här artikeln](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) . 
  
Du kan också köra det [avancerade diagnostikverktyget](https://aka.ms/SaRA-OutlookAdvDiagnostics) på datorn.
