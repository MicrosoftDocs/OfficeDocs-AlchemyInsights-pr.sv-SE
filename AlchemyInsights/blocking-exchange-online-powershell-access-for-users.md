# <a name="blocking-exchange-online-powershell-access-for-users"></a>Blockera Exchange Online PowerShell-åtkomst för användare
Som administratör kan du blockera åtkomst till Exchange Online PowerShell för dina användare. Mer information finns i följande avsnitt:

- [Aktivera eller inaktivera åtkomst till Exchange Online PowerShell](https://docs.microsoft.com/powershell/exchange/exchange-online/disable-access-to-exchange-online-powershell)

- [Klient åtkomst regler i Exchange Online](https://technet.microsoft.com/library/mt842508.aspx) (Sök efter **RemotePowerShell** -protokollet) 

**Obs!** se till att inte blockera åtkomst till Exchange Online PowerShell för administratörer.
