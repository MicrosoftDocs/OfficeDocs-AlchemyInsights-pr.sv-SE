# <a name="sharepoint-migration-assessment-tool-smat"></a>SharePoint Migration Assessment Tool (SMAT)

Utvärderings verktyget för SharePoint-migreringen (SMAT) är en enkel kommando rads program fil som söker igenom innehållet i SharePoint-servergruppen för att identifiera hur du kan migrera servern till SharePoint Online med hjälp av Microsoft 365.

[Översikt över SharePoint Migration Assessment Tool](https://docs.microsoft.com/sharepointmigration/overview-of-the-sharepoint-migration-assessment-tool)
