# <a name="block-pop-up-windows-in-microsoft-edge"></a>Blockera popup-fönster i Microsoft Edge

1. Gå till **... (Inställningar och mer)**  >  **Inställningar**  >  för **Webbplats behörigheter**.
2. Välj **pop-ups och omdirigeras**.
3. Aktivera **blockera**.
