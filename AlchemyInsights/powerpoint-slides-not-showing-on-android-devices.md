# <a name="powerpoint-slides-not-showing-in-skype-for-business-on-android-devices"></a>PowerPoint-bilder visas inte i Skype för företag på Android-enheter

Om det uppstår problem med att PowerPoint-bilder inte visas på Android-enheter kan det bero på ett fel i webbläsaren Chrome.

Prova att inaktivera och sedan avinstallera Chrome. När Chrome har avinstallerats aktiverar du Chrome igen för att hämta fabriksversionen (74.0.3729.136) där problemet är löst.
