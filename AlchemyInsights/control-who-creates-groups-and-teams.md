# <a name="control-who-creates-microsoft-365-groups-and-microsoft-teams"></a>Styra vem som skapar Microsoft 365-grupper och Microsoft Teams

Som standard kan alla användare inom organisationen skapa Microsoft 365-grupper och Microsoft Teams. Du kan ändra standardinställningen och tilldela rätten att skapa Microsoft 365-grupper eller Teams till en delmängd användare inom organisationen. Den här ändringen påverkar också skapandet av grupper från alla Microsoft 365-tjänster som Outlook, Teams, Planner och så vidare.

Mer information finns i [Hantera vem som kan skapa Microsoft 365-grupper](https://support.office.com/article/Manage-who-can-create-Office-365-Groups-4c46c8cb-17d0-44b5-9776-005fced8e618) och Microsoft [Teams](https://aka.ms/rtsf).
