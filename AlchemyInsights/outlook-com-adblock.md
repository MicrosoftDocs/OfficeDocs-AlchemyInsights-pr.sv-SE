# <a name="adblock"></a>Adblock

Vissa användare har för närvarande problem när de läser eller skriver e-post när de använder Google Chrome med AdBlock-tillägget. Så här kan du lösa problemet:

1. Använd [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) eller en annan webbläsare.
1. Uteslut Outlook.com från AdBlock-inställningarna.Längst upp till höger i Google Chrome väljer du **AdBlock** och väljer sedan **Kör inte på sidor på den här domänen**.Uppdatera webbläsaren så att Outlook.com laddas igen.
