# <a name="microsoft-stream-licensing-overview"></a>Översikt över Microsoft Stream-licensiering

Microsoft Stream är en molntjänst som licensieras per användarprenumeration. Den är tillgänglig som en del av berättigade Office 365-prenumerationer. Mer information om varje prenumerationstyp finns i Office 365-priser.

- [Översikt över Microsoft Stream-licensiering](https://docs.microsoft.com/stream/license-overview)
