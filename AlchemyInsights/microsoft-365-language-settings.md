# <a name="microsoft-365-language-settings"></a>Språkinställningar för Microsoft 365

Ta reda på om du vill ändra visningsspråket och andra inställningar för användare eller för hela klienten.

- För användare finns [i Ändra visningsspråk och tidszon i Microsoft 365 för företag för alla appar](https://support.microsoft.com/office/6f238bff-5252-441e-b32b-655d5d85d15b).
- För klienten finns i [Ange språkinställningar och nationella inställningar för Microsoft 365](https://docs.microsoft.com/office365/troubleshoot/access-management/set-language-and-region).
