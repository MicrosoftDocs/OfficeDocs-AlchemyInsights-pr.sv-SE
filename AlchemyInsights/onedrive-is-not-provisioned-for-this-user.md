# <a name="onedrive-is-not-provisioned-for-this-user"></a>OneDrive har inte etablerats för den här användaren

För nya konton måste användarna logga in i administrations centret för Microsoft 365 med sitt eget konto och klicka på OneDrive för att manuellt starta tillhandahållandet av OneDrive-webbplatsen.
  
Om användaren redan har klickat på OneDrive och deras personliga webbplats inte tillhandahålls inom 24 timmar, loggar du in ett support ärende för att få hjälp.
