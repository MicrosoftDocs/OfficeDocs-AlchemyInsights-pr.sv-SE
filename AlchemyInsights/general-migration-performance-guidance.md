# <a name="general-migration-performance-guidance"></a>Allmän prestandavägledning för migrering


**Viktigt**: under denna extraordinära tid vidtar vi åtgärder för att se till att SharePoint Online- och OneDrive-tjänsterna har hög tillgänglighet. Mer information finns i [temporära funktionsjusteringar för SharePoint Online](https://aka.ms/ODSPAdjustments).

**Migreringsalternativ**

Det finns olika alternativ som du kan använda för att migrera innehåll till SharePoint Online beroende på storleken och antalet filer som måste flyttas. Du hittar en lista med alternativ [här](https://docs.microsoft.com/sharepointmigration/migrate-to-sharepoint-online).

Om du vill ha mer information om att migrera innehåll kan du gå till länkarna nedan.

- [Migreringsverktyget för SharePoint](https://docs.microsoft.com/sharepointmigration/introducing-the-sharepoint-migration-tool)

- [Komma igång med Migreringshanteraren](https://docs.microsoft.com/sharepointmigration/mm-get-started)

- [Mover](https://mover.io/)
