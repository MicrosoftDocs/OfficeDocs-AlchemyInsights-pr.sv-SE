# <a name="steps-to-resolve-office-activation-error-0x80072efd"></a>Anvisningar för att lösa fel i Office-0x80072EFD

Prova de här lösningarna för att åtgärda Office Activation-felet:
  
1. Kontrol lera [brand väggen](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380#BKMK_CheckFirewall).

2. Kontrol lera [antivirus programmet](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380#BKMK_CheckAV).

3. Kontrol lera dina [proxyinställningar](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380#BKMK_CheckProxy)

4. Ladda ned och installera [support-och återställnings assistenten](https://aka.ms/SARA-OfficeActivation-Alchemy) och kör **fel sökaren för aktivering**.
