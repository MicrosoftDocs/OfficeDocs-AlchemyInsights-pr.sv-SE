# <a name="steps-to-resolve-office-activation-error-0x80072efd"></a>Åtgärder för att lösa aktiveringsfel för Office 0x80072EFD

Prova de här lösningarna för att åtgärda aktiveringsfelet för Office:
  
1. Kontrollera [brandväggen](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380#BKMK_CheckFirewall).

2. Kontrollera [antivirusprogrammet](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380#BKMK_CheckAV).

3. Kontrollera [proxyinställningarna](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380#BKMK_CheckProxy)

4. Hämta och installera [support- och återställningsassistenten](https://aka.ms/SARA-OfficeActivation-Alchemy) och kör **felsökaren för aktivering**.
