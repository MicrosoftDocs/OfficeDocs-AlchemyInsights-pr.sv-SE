# <a name="sharepoint-publishing-infrastructure"></a>Infrastruktur för SharePoint-publicering


SharePoint innehåller en klassisk uppsättning funktioner som gör att du kan redigera och publicera rika webb sidor för intranätet. Aktivera infrastrukturen för SharePoint-publicering på webbplats samlings nivån för att aktivera funktionen.

Mer information finns på länkarna nedan.

[Aktivera publicerings funktioner](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[Publicerings funktioner](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

Kommentarer:

- De klassiska mallarna för publicerings webbplatser innehåller funktionerna för publicering som standard.

- Klassiska publicerings funktioner är inte tillgängliga på moderna webbplatser. Moderna webbplatser tillhandahåller ett nyare alternativ som standard.
