# <a name="sharepoint-publishing-infrastructure"></a>SharePoint-publicering infrastruktur


<p>SharePoint innehåller en klassisk uppsättning funktioner som gör att du kan redigera och publicera innehållsrika webbsidor för intranätet. Aktivera SharePoint Publishing infrastruktur på webbplatssamlingsnivå om du vill aktivera dem.</p>  <p>Mer information finns på länkarna nedan.</p>  <ul>  <li><a href="https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518">Aktivera publiceringsfunktioner</a></li>  <li><a href="https://support.office.com/en-us/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553">Publiceringsfunktionerna</a></li>  </ul>  <p>Anteckningar:</p>  <ul>  <li>De klassiska publishing mallarna innehåller publishing funktioner som standard.</li>  <li>Klassiska Publiceringsfunktionerna är inte tillgängliga på moderna webbplatser. Moderna webbplatser är en nyare alternativ som standard.</li>  </ul>
