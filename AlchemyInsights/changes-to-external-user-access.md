# <a name="changes-to-external-user-access"></a>Ändringar i extern användaråtkomst

Från och med den 23 mars 2018 kan externa användare inte längre komma åt innehåll som har delats med alla, alla autentiserade användare eller alla formuläranvändare, även om de har bjudits in eller lagts till i behöriga grupper. Externa användare kan bara komma åt innehåll som har delats med användaren eller med grupper som användaren tillhör. Om du vill behålla dina befintliga inställningar, se:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
