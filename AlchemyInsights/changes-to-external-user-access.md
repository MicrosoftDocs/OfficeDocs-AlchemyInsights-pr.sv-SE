# <a name="changes-to-external-user-access"></a>Ändringar av extern användaråtkomst

Från och med 23 mars 2018 kommer externa användare inte längre att kunna komma åt innehåll som har delats med alla, alla autentiserade användare eller alla formuläranvändare, även om de har bjudits in eller lagts till i auktoriserade grupper. Externa användare kommer endast att kunna komma åt innehåll som har delats med användaren eller med grupper som användaren tillhör. Om du vill behålla dina befintliga inställningar, vänligen se:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
