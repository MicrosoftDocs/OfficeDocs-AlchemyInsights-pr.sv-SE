# <a name="changes-to-external-user-access"></a>Ändringar av åtkomst för externa användare

Startar den 23 mars 2018 kommer externa användare inte längre kunna komma åt innehåll som har delats med alla, alla, Autentiserade användare eller alla användare formulär, även om de har bjudit in eller läggas till i auktoriserade grupper. Externa användare ska kunna komma åt innehåll som har delats med användaren eller grupper som användaren tillhör. Om du vill behålla dina befintliga inställningar finns:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
