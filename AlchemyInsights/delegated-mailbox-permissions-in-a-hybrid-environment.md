# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>Delegerade postlådan behörigheter i en hybrid-miljö

Beroende på vilken version av Exchange som du har installerat i organisationen på plats kan du behöva vidta några extra åtgärder för att konfigurera delegerade postlådan behörigheter mellan din lokala organisation och Exchange Online i Office 365. Mer information om hur du ställer in delegerad behörighet att ta en titt på [Konfigurera Exchange för att stödja delegerade postlådan behörigheter i en hybrid-distribution](https://technet.microsoft.com/library/mt784505%28v=exchg.150%29.aspx).
