# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>Behörigheter för delegerade post lådor i en hybrid miljö

Beroende på vilken version av Exchange du har installerat i din lokala organisation kan du behöva vidta ytterligare åtgärder för att konfigurera behörigheter för delegerade post lådor mellan din lokala organisation och Exchange Online i Microsoft 365. Mer information om hur du konfigurerar delegerade behörigheter finns [i Konfigurera Exchange för att hantera behörigheter för delegerade post lådor i en hybrid distribution](https://technet.microsoft.com/library/mt784505%28v=exchg.150%29.aspx).
