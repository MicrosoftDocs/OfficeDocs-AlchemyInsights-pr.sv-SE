# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>Delegerade postlådebehörigheter i en hybridmiljö

Beroende på vilken version av Exchange du har installerat i din lokala organisation kan du behöva vidta några extra åtgärder för att konfigurera delegerade postlådebehörigheter mellan din lokala organisation och Exchange Online i Microsoft 365. Mer information om hur du konfigurerar delegerade behörigheter finns i [Konfigurera Exchange för att stödja delegerade postlådebehörigheter i en hybriddistribution](https://technet.microsoft.com/library/mt784505%28v=exchg.150%29.aspx).
