# <a name="a-user-has-two-mailboxes"></a>En användare har två postlådor

Hybrid-miljöer som använder Azure Active Directory ansluta (Ledsagardokumentet ansluta) eller DirSync oavsiktligt kan orsaka användaren har två postlådor: en i lokal organisation och en i molnet.

Lös det här problemet finns i [Permanent Radera tidigare postlåda Info](https://blogs.technet.microsoft.com/exchange/2018/01/17/permanently-clear-previous-mailbox-info/).
