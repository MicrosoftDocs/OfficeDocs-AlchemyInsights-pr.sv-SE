# <a name="client-context---no-enrolled-devices"></a>Klientkontext-inga registrerade enheter

Din klientorganisation har inga MDM-registrerade enheter (Mobile Device Management) med Intune. Om du vill registrera och hantera dina enheter via Intune MDM läser du dokumentationen [här](https://docs.microsoft.com/intune/device-enrollment) för önskad plattform. Om du använder en tredje parts MDM eller endast Appskyddfunktioner kan ingen åtgärd behövas.
