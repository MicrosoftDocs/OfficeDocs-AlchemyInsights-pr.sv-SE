# <a name="client-context---no-enrolled-devices"></a>Klient kontext-inga registrerade enheter

Klient organisationen har inga registrerade enheter (Mobile Device Management) med Intune. Om du vill registrera och hantera dina enheter via Intune MDM bör du läsa dokumentationen [här](https://docs.microsoft.com/intune/device-enrollment) för önskad plattform. Om du använder MDM från tredje part eller bara program skydd behöver ingen åtgärd vidtas.
