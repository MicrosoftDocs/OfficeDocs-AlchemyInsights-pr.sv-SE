# <a name="using-self-service-sign-up-in-your-organization"></a>Använda självbetjäningsanmäla i din organisation

Med självbetjäning kan registrera sig användare registrera dig för att använda tjänster som betalas av din prenumeration.

- Om du vill styra om användare kan registrera sig för självbetjäningsprenumerationer använder du [azure Set-MsolCompanySettings](https://docs.microsoft.com/powershell/module/msonline/set-msolcompanysettings?view=azureadps-1.0) PowerShell-cmdlet med parametern **AllowAdHocSubscriptions.** Mer information finns i [Hur kontrollerar jag självbetjäningsinställningarna?](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/self-service-purchase-faq?view=o365-worldwide)

- Information om hur du ser alla tillgängliga program finns i [Tillgängliga självbetjäningsprogram](https://docs.microsoft.com/microsoft-365/admin/misc/self-service-sign-up?view=o365-worldwide#available-self-service-programs).

Fullständig information finns [i Använda självbetjäningsanmäla i organisationen](https://docs.microsoft.com/microsoft-365/admin/misc/self-service-sign-up?view=o365-worldwide).
