# <a name="classification-labels"></a>Klassificeringsetiketter

Med känslighets- eller klassificeringsetiketter kan du klassificera och skydda ditt känsliga innehåll, samtidigt som du ser till att användarnas produktivitet och samarbetsförmåga inte hindras.

För mer information, se följande [artikel](https://docs.microsoft.com/microsoft-365/compliance/sensitivity-labels).
