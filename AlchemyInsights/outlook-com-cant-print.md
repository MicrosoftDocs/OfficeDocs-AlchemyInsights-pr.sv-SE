# <a name="unable-to-print"></a>Det går inte att skriva ut

Vi är medvetna om ett problem där vissa användare inte kan skriva ut när de använder Safari eller Edge-webbläsare. Medan vi arbetar på en fix för detta, vänligen använd en annan webbläsare. Vi ber om ursäkt för besväret.
