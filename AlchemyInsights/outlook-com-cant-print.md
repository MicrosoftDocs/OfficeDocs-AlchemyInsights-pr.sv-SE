# <a name="unable-to-print"></a>Det går inte att skriva ut

Vi är medvetna om ett problem där vissa användare inte kan skriva ut med Safari-eller Edge-webbläsare. Medan vi arbetar med en korrigering för detta, Använd en annan webbläsare. Vi ber om ursäkt för besväret.
