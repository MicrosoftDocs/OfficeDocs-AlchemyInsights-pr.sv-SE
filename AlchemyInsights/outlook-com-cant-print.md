# <a name="unable-to-print"></a>Det går inte att skriva ut

Vi är medvetna om ett problem där vissa användare inte kan skriva ut när de använder webbläsaren Safari eller Edge. Medan vi arbetar på en fix för detta, använd en annan webbläsare. Vi ber om ursäkt för besväret.
