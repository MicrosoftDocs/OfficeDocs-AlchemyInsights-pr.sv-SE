# <a name="unable-to-print"></a>Det går inte att skriva ut

Vi är medvetna om ett problem där vissa användare kan inte skriva ut när du använder Safari eller kanten webbläsare. Använd en annan webbläsare medan vi arbetar på att lösa detta. Vi ber om ursäkt för eventuella olägenheter.
