# <a name="proxyaddress-incorrect"></a>ProxyAddress felaktig

När ett objekt synkroniseras till Azure AD, jämförs de värden som anges i attributet proxyAddresses i Active Directory med Azure AD-regler och sedan attributet proxyAddresses fylls i i Azure AD. Därför kan värdena för attributet proxyAddresses för objektet i Active Directory inte vara samma som värdena för attributet proxyAddresses i Azure AD.
  
Om du vill veta mer om hur proxyAddress fylls, se [hur attributet proxyAddress fylls i i Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
