# <a name="proxyaddress-incorrect"></a>Felaktig ProxyAddress

När ett objekt synkroniseras till Azure AD, jämförs värden som anges i attributet proxyAddresses i Active Directory med Azure AD regler och sedan attributet proxyAddresses fylls i Azure AD. Värdena för attributet proxyAddresses för objekt i Active Directory kan därför inte samma som värdet för attributet proxyAddresses i Azure AD.
  
Om du vill veta mer om hur proxyaddress fylls kan se [hur proxyAddress attributet fylls i Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
