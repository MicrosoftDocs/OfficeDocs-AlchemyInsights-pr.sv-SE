# <a name="proxyaddress-incorrect"></a>ProxyAdress felaktigt

När ett objekt synkroniseras med Azure AD jämförs de värden som anges i attributet proxyAddresses i Active Directory med Azure AD-regler och sedan fylls attributet proxyAddresses i Azure AD. Därför kanske attributet proxyAddresses för objektet i Active Directory inte är desamma som värdena för attributet proxyAddresses i Azure AD.
  
Mer information om hur proxyadressen är ifylld finns i Så här [fyller proxyAddress-attributet i Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
