# <a name="proxyaddress-incorrect"></a>ProxyAddress fel

När ett objekt synkroniseras till Azure AD jämförs de värden som anges i attributet proxyAddresses i Active Directory med Azure AD-regler och sedan är proxyAddresses i Azure AD. Därför kan värdena i proxyAddresses-attributet för objektet i Active Directory inte vara samma som värdena för attributet proxyAddresses i Azure AD.
  
Om du vill veta mer om hur proxyAddress är ifyllt kan du läsa [hur proxyAddress-attributet är ifyllt i Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
