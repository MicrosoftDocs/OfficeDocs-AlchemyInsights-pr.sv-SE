# <a name="sharepoint-online-throttling"></a>SharePoint Online-begränsning

SharePoint Online använder begränsning för att bibehålla optimala prestanda och tillförlitlighet i SharePoint Online-tjänsten. Begränsningen begränsar antalet användaråtgärder och samtidiga anrop (med skript eller kod) för att förhindra att resurserna utnyttjas för mycket. 

Mer information om begränsning finns i Undvika [begränsningar och blockeringar i SharePoint Online.](https://docs.microsoft.com/sharepoint/dev/general-development/how-to-avoid-getting-throttled-or-blocked-in-sharepoint-online)

Om du tror att det här felet inte är relaterat till begränsning kan du kontrollera om aktivt underhåll pågår för klientorganisationen genom att navigera till [meddelandecentret.](https://portal.office.com/adminportal/home#/MessageCenter)

 Kontrollera slutligen att du [besöker sidan Tjänstens](https://portal.office.com/adminportal/home#/servicehealth) hälsa för att söka efter rådgivningar/incidenter som kan inträffa.
