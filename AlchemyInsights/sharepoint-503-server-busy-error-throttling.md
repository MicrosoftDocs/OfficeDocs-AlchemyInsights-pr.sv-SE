# <a name="sharepoint-online-throttling"></a>SharePoint Online-begränsning

<p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Användare kan få ett 503-servern är upptagen fel när du försöker gå till Sharepoint-eller OneDrive.</span></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Det här felet kan orsakas av begränsning i Sharepoint-tjänsten. SharePoint Online använder begränsning för att bibehålla optimal prestanda och tillförlitlighet i SharePoint Online-tjänsten. Begränsning begränsningar anrop antal användaråtgärder eller samtidiga (genom skript eller kod) för att förhindra felaktig användning av resurser. Om du hämta begränsats 99% av tiden är det på grund av anpassad kod.</span></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Mer information om processbegränsning finns <a href="https://docs.microsoft.com/en-us/sharepoint/dev/general-development/how-to-avoid-getting-throttled-or-blocked-in-sharepoint-online">Undvik komma begränsats eller blockeras i SharePoint Online</a>.</span></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Om du tror att det här felet är beroende av begränsning kan du kontrollera om det finns aktiva Underhåll uppstår på din hyresgäst genom att navigera till <a href="https://portal.office.com/adminportal/home#/MessageCenter">Message center</a>. Slutligen se till att du besöker sidan <a href="https://portal.office.com/adminportal/home#/servicehealth">Service hälsa</a> för alla rekommendationerna/olyckor som kan ske.</span></p> <p>&nbsp;</p>
