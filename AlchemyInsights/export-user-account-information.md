# <a name="export-user-account-information"></a>Exportera information om användarkonton

Om du vill exportera användarkontoinformation i Microsoft 365-administrationscentret, inklusive information om licenser, lösenordsprincip, blockerade konton, AAD-synkroniseringsstatus med mera, går du till sidan Aktiva[användare](https://go.microsoft.com/fwlink/p/?linkid=834822) för användare av **Användare** > och väljer sedan **Exportera användare**.
