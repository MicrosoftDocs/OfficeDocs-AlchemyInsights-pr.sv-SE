# <a name="remove-your-godaddy-domain-from-microsoft-365"></a>Ta bort godaddy-domänen från Microsoft 365

Vi kan inte ta bort din GoDaddy-domän från Microsoft 365 åt dig. Ring [GoDaddys kundtjänst](https://aka.ms/contact-godaddy) för att få hjälp.
  
Om organisationsnamnet inte innehåller "NETORG" eller inte har köpts från GoDaddy tar du bort domänen genom att följa dessa steg [Ta bort en domän från Microsoft 365.](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/remove-a-domain)
