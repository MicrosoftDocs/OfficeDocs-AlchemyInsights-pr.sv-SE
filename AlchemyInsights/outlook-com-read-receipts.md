# <a name="read-receipts"></a>Läs kvitton

Det går inte att begära Läs kvitto i Outlook.com, men du kan begära Läs kvitton för Outlook.com meddelanden som skickas med Outlook för Windows. Mer information finns i [lägga till och begära Läs kvitton och leverans meddelanden](https://support.office.com/article/a34bf70a-4c2c-4461-b2a1-12e4a7a92141?wt.mc_id=Office_Outlook_com_Alchemy).
  
Om du vill välja hur Outlook.com reagerar på begäran om Läs kvitton går du till [Läs kvitto inställningar](https://outlook.live.com/mail/options/mail/handling/readReceipts).
