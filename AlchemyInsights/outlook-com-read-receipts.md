# <a name="read-receipts"></a>Läskvitton

Du kan inte begära ett läskvitto i Outlook.com, men du kan begära läskvitton för Outlook.com meddelanden som skickas med Outlook för Windows. Mer information finns i [Lägga till och begära läskvitton och leveransmeddelanden](https://support.office.com/article/a34bf70a-4c2c-4461-b2a1-12e4a7a92141?wt.mc_id=Office_Outlook_com_Alchemy).
  
Om du vill välja hur Outlook.com svarar på läskvittoförfrågningar går du till Inställningarna för [läskvitto](https://outlook.live.com/mail/options/mail/handling/readReceipts).
