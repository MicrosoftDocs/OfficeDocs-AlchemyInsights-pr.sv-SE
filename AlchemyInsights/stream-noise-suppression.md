# <a name="stream-noise-suppression"></a>Brusreducering för ström

Brusreducering, när den slås på av videoägare, gör det möjligt för tittarna att isolera tal från andra ljud och musik under uppspelning. Brusreducering kan slås på eller stängas av för varje nyligen uppladdad video. Mer information finns i [Brusreducering](https://docs.microsoft.com/stream/noise-suppression).
