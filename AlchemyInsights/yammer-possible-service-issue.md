# <a name="yammer---possible-service-issue"></a>Yammer - möjliga problem

Det kan vara ett tillfälligt problem där användare kan uppstå fel vid åtkomst till Yammer. Kontrollera att [Tjänsten hälsa instrumentpanelen](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) för att se om detta påverkar nätverket.
