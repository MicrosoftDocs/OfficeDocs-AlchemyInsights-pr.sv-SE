# <a name="yammer---possible-service-issue"></a>Yammer – möjligt tjänst problem

Det kan finnas ett tillfälligt problem där användarna kan få fel meddelanden vid åtkomst till Yammer. Kontrol lera [instrument panelen för tjänstens hälsa](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) för att se om det påverkar ditt nätverk.
