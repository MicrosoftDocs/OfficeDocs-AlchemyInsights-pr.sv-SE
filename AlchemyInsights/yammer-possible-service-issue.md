# <a name="yammer---possible-service-issue"></a>Yammer-möjliga service problem

Det kan finnas ett tillfälligt problem där användare kan uppstå fel vid åtkomst till Yammer. Kontrollera den [service Health instrumentpanelen](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) för att se om detta påverkar ditt nätverk.
