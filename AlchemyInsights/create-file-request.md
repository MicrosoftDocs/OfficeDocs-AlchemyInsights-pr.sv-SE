# <a name="how-to-create-a-file-request"></a>Så här skapar du en filbegäran

Med funktionen för fil förfrågan i OneDrive kan du välja en mapp där andra kan ladda upp filer med hjälp av en länk som du skickar dem till. Användarna som du efterfrågar filer från kan inte se vad som finns i mappen – de kan bara ladda upp filer till den.

**Obs!** en administratör måste aktivera [alla länkar](https://docs.microsoft.com/sharepoint/turn-external-sharing-on-or-off) i OneDrive för att fil förfrågningar ska fungera.

- Mer information finns i [skapa en filbegäran](https://support.office.com/article/create-a-file-request-f54aa7f8-2589-4421-b351-d415fc3b83af).
