# <a name="how-to-create-a-file-request"></a>Så här skapar du en filförfrågan

Med funktionen för filförfrågan på OneDrive kan du välja en mapp där andra kan ladda upp filer med hjälp av en länk som du skickar dem till. De användare som du begär filer från kan inte se vad som finns i mappen-de kan bara ladda upp filer till den.

[Skapa en filbegäran](https://support.office.com/article/create-a-file-request-f54aa7f8-2589-4421-b351-d415fc3b83af)
