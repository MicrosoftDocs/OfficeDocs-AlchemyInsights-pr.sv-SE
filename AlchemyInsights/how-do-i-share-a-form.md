# <a name="share-a-form"></a>Dela ett formulär

Om du vill dela ett frågeformulär eller formulär i Microsoft Forms finns i [Dela ett formulär för att samarbeta](https://support.office.com/article/Share-a-form-to-collaborate-d5bb5cf0-8401-4c15-bb8c-8e108cd7e69b).

Mer om du vill hantera möjligheten att dela formulär externt finns i [Konfigurera Microsoft Forms](https://support.office.com/article/set-up-microsoft-forms-cc52287a-4550-464d-9a1b-457bf9df2240). 

Så här får du en sammanfattningslänk för att dela resultat med andra:

1. Klicka **Responses** på ellipsknappen (**...**).
3. Välj **Skapa en sammanfattningslänk**.
4. Klicka på **Kopiera**.

Alla som du delar den här länken med kan visa en sammanfattning av svaren för formuläret.
