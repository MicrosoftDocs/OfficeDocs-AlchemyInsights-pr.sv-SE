# <a name="delete-a-sharepoint-site"></a>Ta bort en SharePoint-webbplats

1. I det nya administrationscentret för SharePoint klickar du på kryssrutan bredvid den webbplats som du vill ta bort på sidan webbplatser.
    
2. Klicka på **ta bort**i verktygsfältet menyfliksområdet.
