# <a name="fix-authentication-failed-errors-in-microsoft-flow"></a>Åtgärda fel i autentiseringen misslyckades i Microsoft Flow

Om felmeddelandet "Autentisering misslyckades" visas när du försöker logga in på Microsoft Flow beror det på att Flow-tjänsten är inaktiverad i Azure Active Directory. Mer information och åtgärder för att åtgärda felet finns [i Du kan inte logga in på Microsoft Flow och felet "Autentisering misslyckades".](https://support.microsoft.com/help/4316891)
