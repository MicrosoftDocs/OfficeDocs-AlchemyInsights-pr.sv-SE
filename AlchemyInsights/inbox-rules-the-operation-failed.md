# <a name="rules-and-alerts-error-the-operation-failed"></a>Regler och aviseringar fel: ”åtgärden misslyckades”

Ett Inkorgen regler-felmeddelande kan visas när du har installerat följande uppdateringar:
- [Outlook 2016: Säkerhetsuppdatering November KB](https://support.microsoft.com/help/4461506)
- [Outlook 2013: Säkerhetsuppdatering November KB](https://support.microsoft.com/help/4461486)
- [Outlook 2010: Säkerhetsuppdatering November KB](https://support.microsoft.com/help/4461585) 

Du kan se senaste status för detta kända problem genom att besöka [felloggen för Outlook: ”åtgärden misslyckades...” När du väljer hantera regler & varningar](https://support.office.com/en-us/article/Outlook-Error-The-operation-failed-when-selecting-Manage-Rules-Alerts-64b6ff77-98c2-4564-9cbf-25bd8e17fb8b%20).
