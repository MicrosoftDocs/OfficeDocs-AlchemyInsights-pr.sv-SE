# <a name="migrating-email-from-imap-email-system-to-microsoft-365"></a>Migrera e-post från IMAP-e-postsystem till Microsoft 365

Följ den här proceduren om du vill migrera e-post från ett IMAP-e-postsystem: [IMAP-migrering i administrations centret för Microsoft 365](https://docs.microsoft.com/Exchange/mailbox-migration/migrating-imap-mailboxes/imap-migration-in-the-admin-center)
  
Många stora organisationer måste planera omsorgsfullt för att hantera användar förväntningar. Det här [behöver du veta om migrering av IMAP-postlådor till Microsoft 365](https://docs.microsoft.com/Exchange/mailbox-migration/migrating-imap-mailboxes/migrating-imap-mailboxes) -artikel hjälper dig att hantera ändringen. Observera att G Suite (Google)-migreringar [inte längre behöver göras med IMAP](https://docs.microsoft.com/Exchange/mailbox-migration/perform-g-suite-migration).

Du kan också gå till sidan [anta Microsoft 365 med FastTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) om du behöver hjälp med migreringen.
