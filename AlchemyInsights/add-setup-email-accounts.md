# <a name="set-up-email-accounts"></a>Konfigurera e-postkonton

1. Klicka på **fil**  >  **information**  >  **Lägg till konto**i Outlook.

2. Ange din e-postadress och klicka sedan på **Anslut**.

3. Ange ditt lösen ord och klicka på **OK**.

4. Du kan upprepa de här stegen för att lägga till flera konton.

**Obs!** vissa e-postkonton, till exempel Gmail, Yahoo och iCloud, kräver att du konfigurerar tvåfaktorautentisering för att kunna använda Outlook för att skicka och ta emot e-post. Mer information finns i [lägga till ett e-postkonto i Outlook](https://support.office.com/article/6e27792a-9267-4aa4-8bb6-c84ef146101b.aspx) .
