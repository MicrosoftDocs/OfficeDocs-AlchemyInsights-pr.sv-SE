# <a name="uk-domain-transfers"></a>. uk domän överföringar

ontact Microsoft support och begära att en **IP-märkning ändras** så att den matchar den registrator som du vill hantera din domän. När märket ändras överförs domänen direkt till den nya registratorn. Sedan måste du arbeta med den nya registratorn för att slutföra överföringen, som troligen betalar överförings avgifter och lägga till den överförda domänen på ditt konto hos din nya registrator.
