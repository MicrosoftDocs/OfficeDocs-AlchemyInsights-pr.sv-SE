# <a name="data-loss-prevention-dlp-overview"></a>Data förlust Prevention (DLP) översikt

En DLP-princip kan du:

- Identifiera känslig information på många platser som Exchange Online, SharePoint Online och OneDrive för företag.


Till exempel kan du identifiera alla dokument som innehåller ett kreditkortsnummer lagras i en OneDrive för Business-webbplatsen eller du kan övervaka bara vissa personer OneDrive platser.

- Förhindra oavsiktlig delning av känslig information.


Läs följande [artikel](https://docs.microsoft.com/en-us/office365/securitycompliance/data-loss-prevention-policies)för mer information.
