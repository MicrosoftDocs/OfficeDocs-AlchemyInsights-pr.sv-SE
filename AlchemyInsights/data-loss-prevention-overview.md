# <a name="data-loss-prevention-dlp-overview"></a>Översikt över data förlust skydd (DLP)

Med en DLP-princip kan du:

- Identifiera känslig information på många platser, till exempel Exchange Online, SharePoint Online och OneDrive för företag.


Du kan till exempel identifiera alla dokument som innehåller ett kreditkorts nummer som lagras på valfri OneDrive för företag-webbplats, eller så kan du bara övervaka OneDrive-webbplatserna för vissa personer.

- Förhindra oavsiktlig delning av känslig information.


Mer information finns i följande [artikel](https://docs.microsoft.com/microsoft-365/compliance/data-loss-prevention-policies).
