# <a name="create-custom-content-types"></a>Skapa anpassade innehållstyper

1. Markera listan som du vill arbeta med i Snabbstart, eller klicka på **Inställningar** (kugghjulsikon) och klicka sedan på **Webbplatsinställningar**.
2. Klicka på **webbplatsinnehållstyper** under **Webbdesignergallerier**.
3. Klicka på Skapa på sidan Webbplatsinnehållstyper.
4. På sidan Ny webbplatsinnehållstyp fyller du i informationen för din anpassade innehållstyp.

Mer information finns i [Anpassade innehållstyper](https://support.microsoft.com/office/e1277a2e-a1e8-4473-9126-91a0647766e5#__toc323548991).
