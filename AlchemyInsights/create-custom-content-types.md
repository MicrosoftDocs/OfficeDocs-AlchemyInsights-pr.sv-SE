# <a name="create-custom-content-types"></a>Skapa anpassade innehålls typer

1. Välj den lista du vill arbeta med i snabb start eller klicka på **Inställningar**  (kugg hjuls ikonen) och klicka sedan på  **webbplats inställningar**.
2. Klicka på **webbplats innehålls**  typer under  **Web Designer-gallerier**.
3. På sidan webbplats innehålls typer klickar du på Skapa.
4. På sidan ny webbplats innehålls typ fyller du i informationen för din anpassade innehålls typ.

Mer information finns i  [anpassade innehålls typer](https://support.microsoft.com/office/e1277a2e-a1e8-4473-9126-91a0647766e5#__toc323548991).
