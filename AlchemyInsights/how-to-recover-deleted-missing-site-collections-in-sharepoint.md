# <a name="recover-missing-or-deleted-site-collections"></a>Återställa saknade eller borttagna webbplatssamlingar

Mer information om hur du återställer saknade eller borttagna webbplatssamlingar i SharePoint finns i följande artiklar.

[Återställa borttagna webbplatser](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection)

[Återställa med SPO PowerShell-modul](https://support.office.com/article/Introduction-to-the-SharePoint-Online-Management-Shell-C16941C3-19B4-4710-8056-34C034493429)

Om du inte hittar webbplatsen kanske du vill [söka i granskningsloggen i Microsoft 365 Security &amp; Compliance Center](https://docs.microsoft.com/microsoft-365/compliance/search-the-audit-log-in-security-and-compliance).

**Anm.:** "Moderna" gruppwebbplatser visas inte i det **klassiska administrationsgränssnittet** för SharePoint. Du kan komma åt listan över "moderna" webbplatser från [administrationscentret för Nya Sharepoint](https://docs.microsoft.com/sharepoint/get-started-new-admin-center).
