# <a name="sharepoint-migration-throttling"></a>Begränsning av SharePoint-migrering

SharePoint Online använder begränsning för att bibehålla optimala prestanda och tillförlitlighet i SharePoint Online-tjänsten. Begränsningen begränsar antalet användaråtgärder och samtidiga anrop (med skript eller kod) för att förhindra att resurserna utnyttjas för mycket. För mer information, se:

- [Jag upplever dåliga prestanda eller begränsningar under migreringen](https://docs.microsoft.com/sharepointmigration/sharepoint-online-and-onedrive-migration-speed#faq-and-troubleshooting)
