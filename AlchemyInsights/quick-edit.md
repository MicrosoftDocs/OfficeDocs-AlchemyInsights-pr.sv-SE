# <a name="quick-edit"></a>Snabbredigering

Det finns två sätt att redigera ett objekt i en SharePoint-lista. Du kan redigera ett enskilt objekt i listvyn, vilket är standardmetoden. Eller om du vill vara ännu mer produktiv kan du redigera flera objekt i snabbredigeringsvyn. Du behöver behörighet på Contribute-nivå eller högre för att redigera objekt. Se [Redigera listobjekt i SharePoint Online](https://support.microsoft.com/office/dac1a1c3-a80b-4082-ba57-715cf613d0f7).
