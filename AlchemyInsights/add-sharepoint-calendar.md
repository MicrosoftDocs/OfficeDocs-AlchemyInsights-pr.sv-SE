# <a name="add-a-sharepoint-calendar"></a>Lägga till en SharePoint-kalender

SharePoint-kalendrar är klassiska List typer som ibland förväxlas med Microsoft 365-gruppkalendrar.
 
Följ de här stegen om du vill skapa en SharePoint-kalender:
 
1.  Välj kugg hjuls ikonen **Inställningar** och välj sedan **Lägg till ett program**.
2.  I rutan **Sök efter en app** skriver du **kalender** och trycker på RETUR.
3.  Välj **kalendern**för programmet, och ange sedan ett namn på kalendern.

Mer information om Microsoft 365-grupper finns i [Läs mer om microsoft 365-grupper](https://support.office.com/article/Learn-about-Office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2).
