# <a name="webresource-visibility-property"></a>Webresource Synlighet egendom

Du måste välja alternativet **Synlig som standard** för att webbresursen ska visas i formuläret och vara tillgänglig för användarna. Mer information finns i [Webbresursegenskaper för modelldrivna appformulär](https://docs.microsoft.com/powerapps/maker/model-driven-apps/web-resource-properties-legacy).
