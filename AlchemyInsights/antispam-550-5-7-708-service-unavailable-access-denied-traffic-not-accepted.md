# <a name="550-57708-access-denied-traffic-not-accepted-from-this-ip"></a>550 5.7.708 Åtkomst nekad, trafik som inte accepteras från denna IP

Det här felet uppstår när du skickar e-post från kända IP-adresser med lågt anseende som vanligtvis används av nya kunder.

Om du måste skicka e-post från dessa IP-adresser innan du kan köpa licenser kontaktar du supporten för att begära ett undantag tills du kan köpa licenser.
