# <a name="how-to-type-special-characters-on-a-mac"></a>Så här skriver du specialtecken på en Mac

Om du vill skriva specialtecken på din Mac öppnar du menyn **specialtecken** med **kommandot**tangent bords gen väg för ⌘  +  **Control**  +  **Space**.

En snabb genväg för @ är också **Alt**  +  **64**.
