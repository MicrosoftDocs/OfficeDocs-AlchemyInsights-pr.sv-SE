# <a name="how-to-type-special-characters-on-a-mac"></a>Hur du skriver specialtecken på en Mac

Om du vill skriva specialtecken på din Mac, öppna menyn **specialtecken** med kortkommandot ⌘ **kommandot** + **kontroll** + **utrymme**.

En genväg för @ är också **ALT** + **64**.
