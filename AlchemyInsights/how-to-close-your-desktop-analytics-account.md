# <a name="how-to-close-your-desktop-analytics-account"></a>Så här stänger du ditt Desktop Analytics-konto

Om du konfigurerar Desktop Analytics i din miljö och sedan bestämmer dig för att du behöver ta bort den använder du processen i följande länk:

[Så här stänger du ditt konto](https://docs.microsoft.com/configmgr/desktop-analytics/account-close)

Det första steget är att kontakta Microsoft Support. Öppna ett supportärende för att stänga ditt Desktop Analytics-konto. Fortsätt inte med ytterligare steg förrän du får en bekräftelse på att Microsoft har stängt ditt konto.
