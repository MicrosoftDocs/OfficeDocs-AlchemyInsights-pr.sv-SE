# <a name="how-to-close-your-desktop-analytics-account"></a>Hur du avslutar ditt skrivbord Analytics-konto

Om du ställer in skrivbordet Analytics i din miljö och sedan bestämmer dig för att du måste ta bort den använda processen i följande länk:

[Hur du avslutar ditt konto](https://docs.microsoft.com/sccm/desktop-analytics/account-close)

Det första steget är att kontakta Microsoft Support. Öppna ett supportärende om du vill stänga kontot Desktop Analytics. Inte fortsätta med ytterligare steg tills du får en bekräftelse på att Microsoft stängt ditt konto.
