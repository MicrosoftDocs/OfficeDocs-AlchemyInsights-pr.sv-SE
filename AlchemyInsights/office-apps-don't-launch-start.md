# <a name="microsoft-365-apps-crash-stop-responding-or-dont-launch"></a>Microsoft 365-appar kraschar, slutar svara eller startar inte

Om du vill att Microsoft 365-apparna ska hänga eller stängas av kan du försöka med följande:

- [Undersöka möjliga problem med](https://support.office.com/article/powerpoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d#bkmk_addins) tillägg som installerats för Office

- Verifiera eller [installera de senaste uppdateringarna för Office](https://support.office.com/article/update-office-and-your-computer-with-microsoft-update-2ab296f3-7f03-43a2-8e50-46de917611c5)
- [Reparera Office](https://support.office.com/article/repair-an-office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b) via kontroll panelen
- [Avinstallera](https://support.office.com/article/uninstall-office-from-a-pc-9dd49b83-264a-477a-8fcc-2fdf5dbf61d8) och installera om Office
