# <a name="setting-up-a-hybrid-environment"></a>Konfigurera en hybridmiljö

Om du vill att användarna ska kunna arbeta smidigt tillsammans oavsett om deras data lagras på lokala servrar eller i Microsoft 365 kan du behöva konfigurera en hybridmiljö. Om du vill få en översikt över dina alternativ kan du läsa [Integreringen av Microsoft 365 med lokala miljöer](https://docs.microsoft.com/office365/enterprise/office-365-integration).
