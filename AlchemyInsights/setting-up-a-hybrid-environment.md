# <a name="setting-up-a-hybrid-environment"></a>Konfigurera en hybrid miljö

Om du vill låta dina användare arbeta sömlöst tillsammans oavsett om deras data lagras på lokala servrar eller i Office 365, kan du behöva konfigurera en hybrid miljö. Om du vill få en översikt över dina alternativ kan du checka ut [Office 365-integrering med lokala miljöer](https://docs.microsoft.com/office365/enterprise/office-365-integration).
