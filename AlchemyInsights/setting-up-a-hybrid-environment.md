# <a name="setting-up-a-hybrid-environment"></a>Konfigurera en hybrid miljö

För att låta användarna samar beta på ett sömlöst sätt, oavsett om de lagras på lokala servrar eller i Microsoft 365, kan du behöva konfigurera en hybrid miljö. Om du vill ha en översikt över dina alternativ kan du ta en titt på [Microsoft 365-integreringen med lokala miljöer](https://docs.microsoft.com/office365/enterprise/office-365-integration).
