# <a name="blocking-items-in-your-junk-email-folder"></a>Blockera objekt i mappen skräp post

Om du blockerar ett objekt som redan finns i mappen skräp post tas det bort. Objekt i [mappen skräp post](https://outlook.live.com/mail/junkemail) tas bort automatiskt efter 30 dagar. Markera ett objekt i mappen skräp post för att se när det kommer att tas bort.

Läs mer om hur du [blockerar avsändare eller markerar e-post som skräp post](https://support.office.com/article/a3ece97b-82f8-4a5e-9ac3-e92fa6427ae4).
