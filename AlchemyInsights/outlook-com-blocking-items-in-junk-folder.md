# <a name="blocking-items-in-your-junk-email-folder"></a>Blockera objekt i mappen Skräppost

Om du blockerar ett objekt som redan finns i mappen Skräppost tas det inte bort. Objekt i [mappen Skräppost](https://outlook.live.com/mail/junkemail) tas bort automatiskt efter 30 dagar. Markera ett objekt i mappen Skräppost för att se när det tas bort.

Läs mer om hur du [blockerar avsändare eller markerar e-post som skräppost](https://support.office.com/article/a3ece97b-82f8-4a5e-9ac3-e92fa6427ae4).
