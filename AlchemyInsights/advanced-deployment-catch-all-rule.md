# <a name="deploy-office-apps"></a>Distribuera Office apps

För att distribuera Office apps till flera användare via Office Deployment Tool System Center Konfigurationshanteraren som del av en avbildning av operativsystemet disken eller i ett scenario för delad dator med Windows Server Remote Desktop Services (RDS) eller Terminal Tjänster, bör du använda steg-för-steg-instruktioner i **Office 365 ProPlus klassificering av distribution**.
  
Kontrollera att du har loggat in som en innehavaradministration eller global administration och sedan klicka här att komma igång: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance).
