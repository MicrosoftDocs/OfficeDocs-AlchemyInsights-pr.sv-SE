# <a name="cant-preview-all-of-an-attachment"></a>Det går inte att förhandsgranska alla bifogade filer

Vi är medvetna om ett problem där vissa kunder går inte att förhandsgranska alla bifogade filer. Som en lösning, öppna den bifogade filen i ett nytt fönster.
