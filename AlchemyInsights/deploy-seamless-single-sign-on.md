# <a name="enable-seamless-sso"></a>Aktivera sömlös SSO

Azure Active Directory (Azure AD) sömlös enkel inloggning (sömlös SSO) loggar automatiskt in användare när de är på sina företagsdatorer som är anslutna till företagets nätverk.
  
[Följ stegen i den här artikeln för att aktivera sömlös SSO](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
