# <a name="delete-the-office-document-cache"></a>Ta bort Office Document cache

Om du har fått ett Excel-meddelande om att **en nyare version av filen är tillgänglig** kan du försöka ta bort Office Document cache genom att följa instruktionerna i [ta bort cacheminnet för Office-dokumentet](https://support.office.com/article/b1d3765e-d71b-4bb8-99ca-acd22c42995d). Försök sedan att öppna filen igen.
