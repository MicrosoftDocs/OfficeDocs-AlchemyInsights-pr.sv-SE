# <a name="steps-to-resolve-office-activation-error-0x80070005"></a>Anvisningar för att lösa fel i Office-0x80070005

Om felet 0x80070005 uppstod när du uppdaterade Windows kan du åtgärda problemet genom att uppdatera registret.
  
Ladda ned och installera [support-och återställnings assistenten](https://aka.ms/SARA-OfficeActivation-Alchemy) och kör **fel sökaren för aktivering**.
  
Mer hjälp finns i: [fel kod för Office 0x80070005 när du aktiverar Office](https://support.office.com/article/7aa7600f-df57-4aef-81d2-25509c66f865)
