# <a name="change-the-default-site-language-in-sharepoint-or-onedrive"></a>Ändra standard språk för webbplatsen i SharePoint eller OneDrive 

Du kan inte ändra standard webbplats språk för SharePoint eller OneDrive **efter att webbplatsen har skapats**. Du kan också ändra dina personliga visnings språk inställningar.

Information om hur du ändrar dina personliga inställningar finns i [ändra inställningar för ditt personliga språk och region](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7).

Information om hur du ändrar användar gränssnittet för webbplatsen finns i [välja vilka språk du vill göra tillgängliga för en webbplats användar gränssnitt](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8).
