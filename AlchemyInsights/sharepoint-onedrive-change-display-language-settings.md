# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>Ändra visningsspråkinställningar i SharePoint eller OneDrive 

Du kan inte ändra standardspråkinställningarna för SharePoint eller OneDrive eftersom dessa inställningar definierades när du skapade din Microsoft 365-prenumeration. Du kan dock ändra inställningarna för visningsspråk.

Information om hur du ändrar dina personliga inställningar finns i [Ändra dina personliga språk- och regioninställningar](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7).

Information om hur du ändrar webbplatsens användargränssnitt finns i [Välj de språk som du vill göra tillgängliga för en webbplats användargränssnitt](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8).

Mer information om felsökning av språkinställningar finns i [Du kan inte ändra standardspråket för en SharePoint Online-webbplatssamling](https://docs.microsoft.com/sharepoint/support/sites/change-default-language-site-collection).
