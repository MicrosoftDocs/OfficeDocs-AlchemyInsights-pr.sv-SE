# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>Ändra språkinställningar för visning i SharePoint eller OneDrive 

Du kan inte ändra inställningarna för standardspråk för SharePoint- eller OneDrive, eftersom dessa inställningar definieras när du skapade din Office 365-prenumeration. Du kan dock ändra bildskärmsinställningarna för språket.

Om du vill ändra dina personliga inställningar finns i [ändra dina personliga regioninställningar för språk och](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7).

Om du vill ändra platsens gränssnitt finns i [välja de språk som du vill ska vara tillgängliga för en webbplats användargränssnitt](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8).

Mer information om felsökning av inställningar finns i [du kan inte ändra standardspråket för en SharePoint Online-webbplatssamling](https://support.office.com/article/you-can-t-change-the-default-language-for-a-sharepoint-online-site-collection-40dda07e-6b41-49e9-9828-8805dcb92964).
