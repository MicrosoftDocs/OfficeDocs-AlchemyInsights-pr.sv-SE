# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>Ändra inställningar för visningsspråk i SharePoint eller OneDrive 

Du kan inte ändra standardspråk inställningarna för SharePoint eller OneDrive eftersom dessa inställningar definierades när du skapade din Office 365-prenumeration. Du kan dock ändra inställningarna för visningsspråk.

Om du vill ändra dina personliga inställningar läser du [ändra dina personliga språk-och regionsinställningar](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7).

Om du vill ändra webbplatsens användargränssnitt läser [du Välj de språk som du vill göra tillgängliga för webbplatsens användargränssnitt](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8).

Mer information om hur du felsöker språkinställningar finns [i du kan inte ändra standardspråket för en SharePoint Online-webbplatssamling](https://docs.microsoft.com/sharepoint/support/sites/change-default-language-site-collection).
