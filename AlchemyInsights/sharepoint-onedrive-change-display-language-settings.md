# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>Ändra visnings språk inställningar i SharePoint eller OneDrive 

Du kan inte ändra standard språk inställningarna för SharePoint eller OneDrive eftersom dessa inställningar definierades när du skapade ditt Microsoft 365-abonnemang. Du kan ändra inställningarna för visnings språk.

Information om hur du ändrar dina personliga inställningar finns i  [ändra inställningar för ditt personliga språk och region](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7).

Information om hur du ändrar användar gränssnittet för webbplatsen finns i [välja vilka språk du vill göra tillgängliga för en webbplats användar gränssnitt](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8).

Mer information om hur du felsöker språk inställningar finns i det [går inte att ändra standard språk för en SharePoint Online](https://docs.microsoft.com/sharepoint/support/sites/change-default-language-site-collection)-webbplatssamling.
