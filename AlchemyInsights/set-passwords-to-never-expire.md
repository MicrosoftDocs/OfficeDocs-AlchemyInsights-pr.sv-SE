# <a name="set-passwords-to-never-expire"></a>Här anger du lösenord aldrig upphör att gälla

Ange lösenord aldrig upphör att gälla från [säkerhet &amp; integritet](https://portal.office.com/adminportal/home#/settings/security). Klickar du på ”Redigera” lösenordsprincip och aktivera reglaget till ”on”.
  
Mer detaljerad information finns: [Ange förfallodatum för lösenordsprincipen för din organisation](https://support.office.com/article/set-the-password-expiration-policy-for-your-organization-0f54736f-eb22-414c-8273-498a0918678f).
