# <a name="microsoft-stream-external-sharing"></a>Extern delning i Microsoft Stream

Microsoft Stream stöder ännu inte externa delningsscenarier för gäster eller personer utanför organisationen. Vi vet att dessa funktioner är välbehövliga, och vi arbetar på planer för hur man bygger dessa funktioner.

- [Extern delning i flödet](https://docs.microsoft.com/stream/portal-share-video#external-sharing)

- [Dela dina videoklipp i Microsoft Stream](https://docs.microsoft.com/stream/portal-share-video)
