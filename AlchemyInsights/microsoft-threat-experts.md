# <a name="microsoft-threat-experts"></a>Microsoft Threat Experts

Microsoft Threat Experts är en hanterad tjänst för hot i Microsoft Defender för Endpoint.  Vi samarbetar med ditt Säkerhetscenter för att övervaka, analysera och stödja för att identifiera och hantera kritiska hot i din unika miljö.

Den här tjänsten för hantering av hot ger expertdrivna insikter och data via dessa två funktioner: riktade attackmeddelanden och tillgång till experter på begäran. Se [dokumentationen för Microsoft Threat Experts](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/microsoft-threat-experts) för mer information.
