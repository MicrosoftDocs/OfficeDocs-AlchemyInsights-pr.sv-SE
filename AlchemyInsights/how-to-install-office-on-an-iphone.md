# <a name="how-to-install-office-on-an-iphone"></a>Installera Office på en iPhone

- Få användaren att [trycka på nedladdnings länken](https://support.office.com/article/9df6d10c-7281-4671-8666-6ca8e339b628?wt.mc_id=Alchemy_ClientDIA) från deras mobila enhet för Word, Excel, PowerPoint och OneNote.

- Låt användaren starta appen, tryck på **Logga in** och ange deras arbets-eller skol uppgifter.
