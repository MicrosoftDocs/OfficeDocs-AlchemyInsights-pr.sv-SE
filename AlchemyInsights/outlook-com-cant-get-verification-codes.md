# <a name="cant-get-verification-codes"></a>Kan inte få verifierings koder

Om du inte kan komma åt din alternativa e-postadress eller ditt telefonnummer, gå till [grundläggande](https://account.microsoft.com/security) information om ditt Microsoft-konto, Välj **Jag har inte något av detta** på inloggnings sidan och följ instruktionerna för att ersätta din säkerhets information.

*Obs!* [Microsoft Authenticator-appen](https://go.microsoft.com/fwlink/?linkid=2016117) kräver ingen Internet anslutning för att ge dig en verifierings kod.

Mer information finns i [säkerhets information och verifierings koder för Microsoft-konton](https://support.microsoft.com/help/12428/).
