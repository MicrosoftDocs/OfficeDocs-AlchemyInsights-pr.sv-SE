# <a name="create-users-in-hybrid-deployments"></a>Skapa användare i hybrid distributioner

När din organisation är en hybrid distribution mellan en lokal miljö och Microsoft 365 måste du skapa alla användar objekt i din lokala Active Directory. Du skapar användar objekt i en lämplig organisationsenhet så att de synkroniseras i din organisation.
