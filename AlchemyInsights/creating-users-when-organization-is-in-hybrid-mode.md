# <a name="create-users-in-hybrid-deployments"></a>Skapa användare i hybriddistributioner

När din organisation är en hybriddistribution mellan en lokal miljö och Microsoft 365 måste du skapa alla användarobjekt i den lokala Active Directory. Du skapar användarobjekten i lämplig organisationsenhet så att de synkroniseras i organisationen.
