# <a name="manage-who-can-create-office-365-groups"></a>Hantera vilka som kan skapa Office 365 Grupper

Som standard kan vem som helst i organisationen skapa O365-grupper.
  
Om du vill ändra på det kan du skapa en säkerhetsgrupp med PowerShell som begränsar gruppskapandet till en delmängd användare.
  
Följ steg-för-steg-instruktionerna här: [Hantera vem som kan skapa Office 365-grupper](https://docs.microsoft.com/office365/admin/create-groups/manage-creation-of-groups)
