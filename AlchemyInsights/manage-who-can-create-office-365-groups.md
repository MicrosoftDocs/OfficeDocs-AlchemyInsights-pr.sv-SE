# <a name="manage-who-can-create-office-365-groups"></a>Hantera vilka som kan skapa Office 365 Grupper

Som standard kan alla i organisationen skapa grupper för O365.
  
Om du vill ändra som kan du skapa en säkerhetsgrupp med PowerShell som begränsar skapande av en grupp till en deluppsättning av användare.
  
Följ de stegvisa instruktionerna: [Hantera vem som kan skapa Office 365-grupper](https://support.office.com/article/4c46c8cb-17d0-44b5-9776-005fced8e618)
