# <a name="export-chat-history-in-teams"></a>Exportera chatt i Teams

- Det går för närvarande inte att exportera och arkivera konversations historik för användare direkt från Microsoft Teams. Rösta på den här funktionen i [UserVoice](https://microsoftteams.uservoice.com/forums/555103-public/suggestions/16982542-backup-export-printing-archive-options?page=2&per_page=20).
- Som administratör eller eDiscovery-hanteraren kan du skapa en [innehålls sökning](https://docs.microsoft.com/microsoft-365/compliance/content-search?view=o365-worldwide)  i Microsoft 365 Compliance Center och sedan exportera snabbmeddelandekonversationer för snabb meddelanden i organisationen. Mer information finns i [Exportera innehålls Sök Resultat](https://docs.microsoft.com/microsoft-365/compliance/export-search-results?view=o365-worldwide).
