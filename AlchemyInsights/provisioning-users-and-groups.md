# <a name="provisioning-users-and-groups"></a>Konfigurera användare och grupper

Information om hur du felsöker problem med att etablera användare och grupper finns i fel söknings guiden [inga användare etableras](https://docs.microsoft.com/azure/active-directory/app-provisioning/application-provisioning-config-problem-no-users-provisioned).

Om du upptäcker att användare inte etableras kan du läsa [konfigurations loggar (för hands version)](https://docs.microsoft.com/azure/active-directory/reports-monitoring/concept-provisioning-logs) i Azure Active Directory (AD). Sök efter inloggnings poster för en viss användare.
