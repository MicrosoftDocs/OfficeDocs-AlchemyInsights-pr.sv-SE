# <a name="outlook-on-the-web-account-hacked"></a>Outlook på webbkontot hackat

Om ditt konto har hackats följer du instruktionerna i Så här skyddar och återställer du [e-postfunktionen till ett misstänkt komprometterat Microsoft 365-konto och postlåda](https://docs.microsoft.com/office365/securitycompliance/responding-to-a-compromised-email-account#how-to-secure-and-restore-email-function-to-a-suspected-compromised-office-365-account-and-mailbox).

Mer information om ytterligare åtgärder du kan vidta för att förhindra att ditt konto äventyras finns [i Svara på ett komprometterat e-postkonto i Microsoft 365](https://docs.microsoft.com/office365/securitycompliance/responding-to-a-compromised-email-account).
