# <a name="get-in-app-support-for-the-outlook-mobile-app"></a>Få support i app för Outlook mobile app

1. Öppna **menyn**i appen Outlook mobile.
    
2. Markera ett frågetecken för att öppna den **Hjälp &amp; Feedback** skärmen. 
    
3. Knacka på **Kontakta Support** och förklara problemet. 
    
4. Knacka på **Skicka** och du får ett meddelande i appen när supportpersonalen kontaktar dig. 
    
Detaljerade instruktioner finns i [Hjälp för Outlook för iOS och Android - app](https://support.office.com/article/218a22d1-9fa5-4889-b689-de1c63493243.aspx#ID0EAABAAA=Contact_Support).
