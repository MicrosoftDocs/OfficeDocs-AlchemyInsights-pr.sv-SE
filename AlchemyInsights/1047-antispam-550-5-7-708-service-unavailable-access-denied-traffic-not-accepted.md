# <a name="550-57708-access-denied-traffic-not-accepted-from-this-ip"></a>550 5.7.708 åtkomst nekad, trafik accepteras inte från denna IP-

Det här felet uppstår när du skickar e-post från kända, låg rykte IP-adresser som används oftast av nya kunder.
  
Om du måste skicka e-post från dessa IP-adresser innan du kan köpa licenser, kan du kontakta support för att få ett undantag tills du kan köpa licenser.
