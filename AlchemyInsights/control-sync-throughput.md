# <a name="control-sync-throughput"></a>Styra dataflödet för synkronisering

Om du behöver styra synkroniseringsapptrafiken rekommenderar vi att du använder qos-principerna (Network Quality of Service) eller Windows QoS-principer när det är möjligt. Information om de nätverksinställningar som du kan göra tillgängliga för användarna finns i [Ändra upp- eller nedladdnings- eller nedladdningshastigheten för OneDrive-synkroniseringsappen.](https://support.office.com/article/71cc69da-2371-4981-8cc8-b4558bdda56e)
