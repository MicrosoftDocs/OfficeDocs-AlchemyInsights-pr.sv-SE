# <a name="access-denied-due-to-ip-restriction"></a>Åtkomst nekad på grund av IP-begränsning

Som IT-administratör kan du kontrol lera åtkomsten till SharePoint-och OneDrive-resurser i Microsoft 365 baserat på definierade nätverks platser som du litar på. Detta kallas även platsbaserade principer. Mer information finns i [styra åtkomst baserat på nätverks plats](https://docs.microsoft.com/sharepoint/control-access-based-on-network-location).
