# <a name="copy-and-paste-in-outlookcom"></a>Kopiera och klistra in i Outlook.com

Det finns inga knappar för att kopiera och klistra in i Outlook.com, men du kan använda kortkommandon:

**På en PC:** CTRL + C för att kopiera, CTRL + X för att klippa ut och CTRL + V för att klistra in.

**På en Mac:** CMD + C för att kopiera, CMD + X för att klippa ut och CMD + V för att klistra in.
