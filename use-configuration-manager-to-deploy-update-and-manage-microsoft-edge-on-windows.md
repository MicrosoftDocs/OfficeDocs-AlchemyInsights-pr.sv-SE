# <a name="use-configuration-manager-to-deploy-update-and-manage-microsoft-edge-on-windows"></a>Använda Configuration Manager för att distribuera, uppdatera och hantera Microsoft Edge i Windows

Liksom för Configuration Manager version 1910 kan du distribuera Microsoft Edge (version 77 och senare) med ett PowerShell-skript. Den inaktiverar automatiska uppdateringar och gör att du kan hantera uppdateringarna via Configuration Manager. Mer information om den här och andra fördelarna med Konfigurations hanteraren finns i [Microsoft Edge Management](https://docs.microsoft.com/mem/configmgr/apps/deploy-use/deploy-edge?).
